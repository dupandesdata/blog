Selector pada CSS digunakan untuk menampung property dan value styling agar dapat dipanggil pada atribut html.

Selector All

Selector ini digunakan untuk memberi styling pada semua element html biasa digunakan untuk me-reset semua element.

Syntax selector

* {  

}

Syntax CSS 

* {

    margin: 0;

    padding: 0;

}

Penjelasan:

Kode CSS diatas berfungsi untuk me-reset margin dan padding untuk semua element html, ini berguna supaya kita memiliki kontrol penuh terhadap tag-tag HTML.

Class Selector

Class Selector digunakan sebagai penampung style tertent, class selector harus diawali dengan tanda (titik) di depan nama selector.

Syntax selector

.namaclass { 

}

Syntax CSS 

.myName {

    color: blue;

}

Penjelasan:

Nama class tidak boleh menggunakan spasi hanya boleh menggunakan tanda hubung - contoh:

Syntax CSS 

.my-name {

     color: yellow;

}

Untuk menggunakan class tersebut harus dilakukan pemanggilan dengan atribut class="  "

Syntax HTML 

<p class="myName">Dupandes</p>

<p class="my-name">Milenium</p>

Output 

Dupandes

Milenium

Walaupun memiliki nama selector yang sama tapi memiliki styling yang berbeda, kenapa bisa begitu? Karena nama class selector yang pertama tidak diberi tanda hubung (myName) dan nama class selector yang kedua diberi tanda hubung (my-name). Sesungguhnya kedua class tersebut adalah class yang berbeda walaupun memiliki kemiripan nama.

Class tersebut bisa kita gunakan lebih dari satu tag.

Syntax HTML 

<p class="myName">Dupandes</p>

<p class="my-name">Milenium</p>

<p class="myName">Dani</p>

<p class="my-name">Arifin</p>

ID Selector

Sama seperti class selector cuma dalam penulisan id selector menggunakan tanda # 

Syntax Selector

#namaid { 

}

Syntax CSS 

#prof {

     color: green;

}

Untuk memanggil id selector tersebut dengan menggunakan atribut id=" " pada tag html yang dituju:

Syntax Selector

<p id="prof">Dupandes</p>

Output

Dupandes

Penjelasan:

Jika menggunakan id selector hanya bisa di panggil oleh satu tag saja karena bersifat unik yang hanya 1 didalam dokumen html.

Disarankan jika menggunakan id selector nama id menggunakan dua kata maka kata kedua diawali dengan kapital #myProf karena nanti nya terhubung dengan javascript.

Tag Selector

Selector jenis ini digunakan untuk memberi styling pada tag tertentu:

Syntax selector:

tag { 

}

Syntax CSS 

p {

    color: green;

}

Penjelasan:

Jika menggunakan element selector maka semua element yang menggunakan tag tersebut akan terkena style.

Syntax HTML 

<p>paragraf 1</p>

<p>paragraf 2</p>

<p>paragraf 3</p>

<p>paragraf 4</p>

Output 

paragraf 1

paragraf 2

paragraf 3

paragraf 4

Tag .Class Selector

Selector jenis ini digunakan memberikan styling pada anak dari tag html tersebut.

Syntax Selector 

tag .namaClass {

}

Contoh saya mempunyai dokumen html seperti ini:

Syntax HTML 

<section>

  <p class="Judul">Kurikulum Pendidikan</p>

  <p>Paragraf 2</p>

</section>

<div>

  <p class="Judul">Kurikulum Pendidikan</p>

  <p>Paragraf 2</p>

</div>

Saya ingin memberikan warna red pada class . judul tapi yang hanya berada didalam tag <section> saja.

Syntax CSS

section .Judul {

      color: red;

    }

Output .judul  > section 

Kurikulum Pendidikan

Paragraf 2

Output .judul  > div

Kurikulum Pendidikan

Paragraf 2

Penjelasan:

Syntax CSS diatas berfungsi untuk memberikan styling pada element yang memiliki class . judul yang berada didalam tag <section>, styling tersebut hanya berfungsi jika element class .judul diantara tag <section> dan </section> walaupun memiliki class yang sama jika tidak dibungkus dengan tag <section> maka styling tersebut tidak dapat digunakan.

(Parent) Tag

Select jenis ini digunakan untuk memilih semua tag yang didalam tag parent tersebut.

Syntax Selector

tag (parent) tag {

}

Syntax CSS 

section p {

  color: red;

}

Syntax HTML 

<section>

  <p>Paragraf 1</p>

  <p>Paragraf 2</p>

</section>

<div>

  <p>Paragraf 1</p>

  <p>Paragraf 2</p>

</div>

Penjelasan:

Selector jenis ini digunakan untuk memberi styling pada semua tag <p> yang berada didalam tag parent <section>, styling tersebut hanya berfungsi jika <p> diantara tag <section> dan </section> walaupun memiliki tag yang sama jika tidak dibungkus dengan tag <section> maka styling tersebut tidak dapat digunakan.

Output <p> parent (<section>)

Paragraf 1

Paragraf 2

Output <p> parent (<div>)

Paragraf 1

Paragraf 2

Tag > Tag

Selector jenis ini digunakan untuk memberi styling element yang didalam tag parent.

Syntax Selector

tag (parent) > tag {

}

Syntax CSS

section > p {

  color: green;

}

Syntax HTML

<section>

  <p>Paragraf 1</p>

  <p>Paragraf 2</p>

  <div>

    <p>Paragraf 3</p>

    <p>Paragraf 4</p>

  </div>

</section>

Penjelasan:

Walaupun memiliki tag yang sama namun jika menggunakan selector ini yang mendapatkan styling adalah tag <p> yang memiliki parent <section>. Dalam studi kasus ini element yang memiliki parent <section> adalah paragraf 1 dan paragraf 2, paragraf 3 dan paragraf 4 memiliki parent <div>

Output <p> parent (<section>)

Paragraf 1

Paragraf 2

Output <p> parent (<div>)

Paragraf 4

:first-child

Selector jenis ini digunakan untuk memberi styling pada element anak pertama dari element parent.

Syntax Selector

tag (parent) tag:first-child {

}

Syntax CSS

section p:first-child {

  color: green;

}

Syntax HTML 

<section>

  <p>Paragraf 1</p>

  <p>Paragraf 2</p>

</section>

:nth-child(n)

Selector jenis ini digunakan untuk memilih anak dari parent tertentu.

Syntax Selector 

tag (parent) tag:nth-child(n) {

}

Syntax CSS

section p:nth-child(3) {

  color: green;

}

Syntax HTML 

<section>

  <p>Paragraf 1</p>

  <p>Paragraf 2</p>

  <p>Paragraf 3</p>

  <p>Paragraf 4</p>

</section>

Penjelasan:

(n) diisi dengan nomor anak dari parent pada studi kasus ini saya menuliskan :nth-child(3) yang artinya saya memberikan styling pada<p> yang anak ketiga dari element <section>

Output

Paragraf 1

Paragraf 2

Paragraf 3

Paragraf 4

Syntax Selector

}

}

input:checked

Syntax Selector

input:checked {

}

Syntax CSS 

input:checked {

  height: 50px;

  width: 50px;

}

Syntax HTML 

<form action="">

  <input type="radio" value="wanita" name="gender"> Wanita<br/>

  <input type="checkbox" value="sepeda"> saya punya sepeda<br/>

  <input type="checkbox" value="mobil"> saya punya mobil<br/>

</form>

Penjelasan:

Styling akan berfungsi jika kita mengeklik checkbox.

Output 

Pria
Wanita
saya punya sepeda
saya punya mobil

Grouping Selector 

Selector jenis ini digunakan untuk memberi styling pada lebih dari satu element dengan styling yang sama.

Syntax Selector

tag, tag , tag {

}

Syntax CSS

p, div, a {

  color: green;

}

Syntax HTML

<div>ini element</div>

<p>ini paragraf</p>

<a href="">ini link</a>

Penjelasan:

Kita juga dapat memberikan style multiple selector dengan menggunakan tanda (koma) , di akhir selector tersebut.

Output

ini element

ini paragraf

ini link

Contoh styling multiple

Syntax CSS

p, .link, section .judul, div p:first-child, section p: nth-child(4) {

}
