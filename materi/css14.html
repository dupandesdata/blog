 CSS ini digunakan untuk mempercantik tampilan Unordered List dan Ordered List, Kita dapat mengatur tampilan daftar dengan menggunakan properti daftar CSS sebagai berikut:

  • Membuat jarak antara teks dan penanda dalam daftar.
  • Menggunakan gambar sebagai pengganti angka atau poin.
  • Merubah gaya tampilan bentuk penanda.
  • Mengatur warna background.
  • Menghilangkan penanda.

CSS ini digunakan untuk mengatur element yang memiliki display list-item biasanya <ol> atau <ul> termasuk <li>, karena element <li> ini mewarisi display element induk (<ul> dan <ol>).

Penanda Item Daftar

list-style-type: value;

Properti ini digunakan untuk menentukan gaya penanda item daftar <ul> atau <ol>.

Properti Daftar Posisi

list-style-position: value;

Properti ini digunakan untuk menentukan penanda daftar item terletak di dalam atau di luar konten.

Value:

inside | outside

Penjelasan 

inside penanda akan termasuk kedalam element daftar item jika item berada pada baris kedua, maka akan dibungkus di bawah penanda.

outside ini adalah value default yang menunjukkan bahwa penanda berada di luar element item daftar. 

❛❛ Catatan:

Dalam menggunakan properti list-style-image sebaiknya bersama dengan list-style-type maka jika gambar mengalami masalah sehingga tidak muncul, list-style-type akan ddigunakan sebagai cadangan.
