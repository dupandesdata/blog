Domain Name System (DNS) ditemukan oleh Paul Mackapetris pada tahun 1983 sebelum ditemukan DNS untuk mengakses sebuah website menggunakan file host.txt yang berisi IP address dan nama komputer, namun file ini memiliki kekurangan yaitu saat IP address berubah dan file tersebut ikut berubah.

Apa itu domain?

Domain adalah sebuah nama yang diakhiri dengan .com, .net atau .co.id dll yang terhubung oleh IP address nama tersebut bersifat unik yang terhubung dengan satu IP address tertentu. Contoh google.com saat kita menuliskan domain tersebut maka DNS server kemudian mencari IP address yang terhubung dengan domain tersebut.

Pengertian tentang DNS Server 

DNS server adalah sistem database yang melayani request untuk melakukan pencarian IP address sebuah website dengan menggunakan Internet Protokol atau Transmission Control Protokol  untuk penjelasan tentang protokol baca selengkapnya. 

DNS mempermudah dalam mengakses website jika sebelumnya untuk mengakses website menggunaka IP address misalnya kita ingin mengakses google.com kita harus menuliskan IP address 74.147.224.76 dengan adanya DNS server kita hanya menulis nama domain nya saja (google.com). Maka secara otomatis DNS server kemudian mencari IP address yang dimiliki oleh domain google.com kemudian dikirim ke lokalhost dan di tampilkan peta browser kita.

Jenis DNS Server

Penting untuk mengetahui jenis DNS server karena berguna untuk pemasangan domain nantinya berikut adalah beberapa jenis DNS server:

A record

Jenis DNS ini digunakan untuk menghubungkan hoetname ke IP address 32bit, IP address ini biasa disebut dengan istilah IPv4 contoh sederhana penggunaan A record: nama domain google.com terhubung dengan IP address 74.147.224.76 maka ketika kita menuliskan domain tersebut DNS server akan menghubungkan dengan IP address tersebut.

AAAA record

Jenis DNS ini digunakan untuk menghubungkan hoetname ke IP address 128bit atau biasa disebut dengan istilah IPv6.

CNAME record

Jenis DNS ini digunakan untuk:

  • Memmbuat nama alias domain atau subdomain.
  • Mengarahkan domain ke hosting yang sama.
  • Menghapus nama subdomain misalnya dupancode.blogspot.com menjadi dupancode.my.id
