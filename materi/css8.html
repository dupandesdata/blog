 Border CSS adalah properti CSS yang digunakan untuk memberi atau menghilangkan garis tepi pada element html. 

Border Style

selector {border-style: solid;}

Properti ini digunakan untuk membuat border style (gaya garis) berikut ini adalah value border style:

value Keterangan
none Digunakan untuk menghilangkan border.
hidden sama seperti none.
sotrd Digunakan untuk membuat garis tunggal dan padat seperti bolpoint.
double untuk membuat dua garis seperti bolpoint.
dotted Digunakan untuk membuat garis titik-titik.
dashed Digunakan untuk membuat garis tepi putus-putus.
groove untuk membuat garis terukir.

Border Width

selector {border-width: value;}

Properti ini digunakan untuk mengatur ketebalan garis, value adalah ketebalan garis.

Value Keterangan
px Pixels digunakan untuk memberi ukuran fix pada element/font.
% Persen digunakan untuk memberikan ukuran element/font sesuai dengan lebar/tinggi element parent.

Border Color

selector {border-color: value;}

Properti ini digunakan untuk memberi warna garis tepi, berikut ini adalah value Border color:

Jenis Value
Color red, yellow, green
Hexadecimal #fff, #000, #00ff00
Red-green-blue rgba(108.58.63), rgba(0%.100%.50%)
Hue-saturation-lightness hsl(245.75%.50%)
Hue-saturation-lightness-alpha hsla(245.75%.50%.0.1)

Kita juga bisa mendeklarasikan tiga properti border dalam satu deklarasi seperti ini:

selector {border: 1px solid red;}

Penjelasan:

  • 1px adalah value border width.
  • Solid adalah value border style.
  • Red adalah value border color.

Borde Radius

selector {border-radius: value;}

Properti ini digunakan untuk mengatur sudut element HTML, dengan border radius kita dapat merubah atau mengatur sudut element tertentu value yang digunakan berbentuk persen % dan px.

ini hasilnya border-radius: 25px
ini hasilnya border-radius: 25%

Penjelasan:

Jika kita menggunakan persentase maka sudut akan di buat berdasarkan lebar dari element.

Kita sudah bisa membuat border tetapi border yang kita buat mengelilingi element, supaya kita bisa membuat border pada sisi tertentu bisa menggunakan:

  • border-top: value;
  • border-bottom: value;
  • border-righ: value;
  • border-left:

Source Code

.bdr {border-top: 1px solid red;}

<div class="bdr">ini border-top red</div>
ini border-top red

Penjelasan:

Properti ini menggunakan tiga value yaitu:

  • Border width
  • Border style
  • Border color

Latihan:

Buatlah teks yang memiliki border bottom warna hijau, ketebalan nya 2px dan style double.

Source Code

<style>

 .latihan {

  width: 70%;

  border-bottom: 2px double green;

}

</style>

<div class="latihan">Latihan</div>
Latihan

Kita juga dapat membuat border radius dengan menggunakan top, bottom, right dan left.

  • border-top-right-radius: value;
  • border-top-left-radius value;
  • border-bottom-righ-radius: value;
  • border-bottom-left-radius value;

Penjelasan:

Properti ini digunakan untuk membuat sudut tumpul pada sisi tertentu, value mengikuti border radius diatas.

Latihan:

Buatlah tombol dengan sudut tumpul 20px atas kanan dan 20px bawah kiri.

Source Code 

<style>

  .latihan2 {

    width: 70%;

    padding-left: 13px;

    border: 2px solid green;

    border-top-right-radius: 20px;

    border-bottom-left-radius: 20px;

  }

</style>

<div class="latihan2">Latihan</div>
