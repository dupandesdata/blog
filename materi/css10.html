Text Alignment

Dupandes Milenium KD.

25 November 2022

Properti Text Alignment adalah sebuah properti CSS yang digunakan untuk mengatur penulisan teks horizontal, dalam penempatan teks dapat ditempatkan di kiri atau di kanan, di tengah, atau diratakan.

text-align: value;
Value Keterangan
left Digunakan untuk membuat teks sebaris disejajarkan dengan tepi kiri element
center Digunakan untuk membuat teks berpusat di tengah element
right Digunakan untuk membuat teks sebaris disejajarkan dengan tepi kanan element
justify Digunakan untuk merentangkan setiap baris sehingga smemiliki margin kiri dan kanan lurus (seperti majalah atau koran) kecuali baris terakhir
justify-all Sama halnya dengan justify jika jika ingin meratakan baris terakhir dalam justify

Modifikasi Teks Terakhir

text-align-last

Properti ini digunakan untuk menentukan bagaimana cara baris terakhir di tampilkan.

Value:

right, center dan justify
