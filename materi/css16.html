 Properti CSS yang digunakan untuk mengatur/memanipulasi posisi element tertentu.

position: value;

Terdapat lima jenis value posisi yang berbeda:

static

position: static;

Posisi element secara default posisi ini tidak akan berpengaruh dengan box-offset (top, bottom, right, left dan z-index).

relative

position: relative;

Posisi ini digunakan untuk menaikkan dimensi element kita dapat mengatur box-offset (top, bottom, right, left dan z-index). Namun ketika element diberikan posisi ini tempat yang awal yang tadinya ditempati oleh element tersebut masih tersedia, posisi ini digunakan pada element parent yang memiliki child element absolute. position: absolute;

Posisi absolute jika tidak diberikan parent yang memiliki posisi relative maka ia akan menggunakan body sebagai pembatas.

fixed

position: fixed;

Posisi ini digunakan untuk menetapkan element relative terhadap viewport jika halaman di scroll maka element tersebut ditampilkan melayang,kita dapat mengatur box-offset (top, bottom, right, left dan z-index).

Sticky

position: sticky;

Posisi ini merupakan gabungan dari posisi relative dan fixed, posisi ini bergantung pada user scroll. Element tersebut diposisikan dengan posisi relatif sampai memenuhi offset tertentu terpenuhi di viewport, kemudian jika di scroll akan menempel di tempatnya (seperti posisi: fixed).
