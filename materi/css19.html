 Properti ini digunakan untuk memaksa posisi element berada di kiri atau kanan tetapi tidak keluar dari element parent, float juga terpengaruh oleh box model.

float: none | left | right;
Value Keterangan
none Digunakan untuk menghilangkan float
left Memaksa posisi element berada di kiri
right  Memaksa posisi element berada di kanan 

Untuk memahami tujuan float didalam tata letak cetak, perhatikan gambar berikut:

Element berwarna hijau diatur dengan float left dan element merah diatur dengan float right, sehingga menciptakan ruang kosong kita dapat memberikan teks/element dalam ruang tersebut berbeda dengan menggunakan margin.

Pembersih float

Saat menggunakan float kita mungkin menginginkan element berikutnya berada di bawah (tidak mengambang di kanan atau kiri), dengan menggunakan clear properti.

Clear

clear: value;

Properti ini menentukan apa yang dilakukan oleh element yang berada di sebelah element mengambang (float).

Properti clear dapat memiliki salah satu dari nilai berikut:

Value Keterangan
none Ini default tidak terjadi perubahan.
left Element berada di bawah element float left.
right Element berada di bawah element float right 
both Element berada ke bawah element float left dan right 
❛❛ Catatan:

Dalam membersihkan float, value properti clear harus cocok dengan float yang sedang digunakan, Jika element di-float ke kiri, maka harus menggunakan clear left. 

Clearfix

Terkadang ketika memberikan float pada sebuah element maka element tersebut ditampilkan seperti posisi absolute, ia jika element parent tidak di berikan clearfix maka akan meluap.
