Tag Link

 Tag <a> digunakan untuk membuat hyperlink, yang menghubungkan antara halaman ke halaman lainnya, file, alamat email atau ke website lain.

Tag <a> biasanya memiliki konten yang menunjukkan tujuan link tertentu, atribut terpenting dari tag <a> adalah href atribut tersebut digunakan untuk menyimpan link tujuan.

Syntax HTML

<a href="  ">Konten</a>

Atribut href

Atribut ini berisi URL (Uniform Resource Locator), dalam penulisan URL pada atribut href tidak hanya berbasis HTTP/HTTPS tetapi URL dapat menggunakan dengan skema URL apa pun berikut diantaranya:

  • URL fragmen
  • Potongan fragmen media
  • Nomor telepon yang diawali oleh tel:URL
  • Alamat email yang diawali oleh mailto:URL

Selain menggunakan atribut href tag <a> juga menggunakan atribut sebagai berikut:

Atribut rel

Syntax HTML

<a href="  " rel="  ">Konten/Teks</a>

Atribut ini berisi banyak sekali silahkan cek disini namun terdapat dua jenis value rel yang sering digunakan:

nofollow

Value ini menunjukkan URL bahwa tidak didukung oleh penulis karena penulis tidak memiliki kendali penuh atas URL tersebut.

dofollow

Value ini menunjukkan bahwa penulis mengenai situ url tersebut dan mengikutinya.

Source Code

<a href="https://contoh.my.id/" rel="nofollow">Konten/Teks</a>

<a href="https://dupancode.my.id/" rel="dofollow">Konten/Teks</a>

Atribut target

Syntax HTML

<a href="  " rel="  " target="  ">Konten/Teks</a>

Atribut ini digunakan untuk membuka tab baru ketika diklik berikut beberapa value atribut target yang mempunyai arti khusus:

Source Code

<a href="https://contoh.my.id/" rel="nofollow" target="_blank">Konten/Teks</a>
Nilai Ketarang
_self Value ini adalah penelusuran bawaan membuka link dengan tab yang sama.
_blank  Value ini digunakan untuk membuka tab baru.
_parent Value ini adalah penelusuran induk berperilaku seperti _self.
_top  Value ini adalah penelusuran paling "tertinggi" yang menyatakan leluhur dari konten saat ini, berperilaku seperti _self.
