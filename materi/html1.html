Apa Itu Internet

 Internet merupakan singkatan dari Interconnected Network internet berasal dari proyek lembaga ARPA, proyek ini dimulai pada tahun 1969 tujuan utama dari proyek tersebut adalah untuk menghubungkan antara satu komputer dengan komputer lainnya sebagai kepenting militer US Department of Defense untuk mengantisipasi serangan nuklir.

 Pada tahun 1969 lembaga ARPA menciptakan sebuah jaringan yang diberi nama ARPANET (Advanced Research Project Agency Network), yang mendemonstrasikan hardware dan software komputer berbasis UNIX.

Pada Oktober 1972 proyek ARPA mulai dikenalkan kepada publik karena banyak universitas di negara tersebut yang ingin bergabung untuk dapat menggunakan jaringan ARPANET, maka ARPANET terbagi menjadi dua jenis yaitu "MILNET" digunakan sebagai keperluan militer dan "ARPANET" digunakan untuk keperluan non-militer.

Walaupun terbagi menjadi dua "MILNET" serta "ARPANET" menggunakan jaringan yang diberi nama DARPA Internet, kemudian disederhanakan menjadi Internet.

Pada saat ini internet merupakan sebuah jaringan yang menghubungkan komputer-komputer belahan dunia dengan menggunakan satelit atau kabel bawah laut (fiber optic) dengan menggunakan standar yang ditetapkan yaitu:

  1. Transmission Control Protocol (TCP)
  2. Internet Protocol (IP)
  3. User Datagram Protocol (UDP)

Apa itu protokol?

Protokol adalah sebuah aturan komunikasi untuk pengiriman data antar komputer.

Berikut beberapa protokol yang sering digunakan:

HTTP (Hyper Text Transfer Protokol)

Protokol ini digunakan untuk memberi aturan bagaimana komputer server dan komputer client terhubung. Baca selengkapnya 

FTP (File Transfer Protokol)

Protokol ini digunakan untuk meng-upload atau mendownload file dari komputer client ke server atau sebaliknya. Baca selengkapnya 

SMTP (Simple Mall Transfer Protokol)

Protokol ini digunakan untuk bertukar email dari client ke server sebelum dikirim ke server tujuan. Baca selengkapnya 

Protokol-protokol tersebut digunakan pertukaran paket (packet switching communication protocol) antara berbagai negara dalam arti lintas negara dan lintas benua yang disebut dengan Global Wide Area Network (GWAN). 

www (World Wide Web)

Digunakan untuk mengakses dokumen html  dengan menggunakan protokol HTTP. Baca selengkapnya 

Request & Response

Dalam mengakses sebuah website itu kita harus melakukan request ke server kemudian direspon oleh server dengan cara menampilkan halaman website yang kita request. 

Sessionles

Dalam mengakses website itu memiliki sesi atau masa jadi ketika sesi itu habis akan expired contoh sederhana ketika kita mengakses Facebook dan beberapa hari tidak ada aktivitas yang dilakukan akun Facebook akan keluar dengan sendirinya itu dikarenakan sesinya sudah habis.
