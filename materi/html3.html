Pengenalan Web Hosting

 Web Hosting adalah sebuah website penyedia layanan hosting, website tersebut digunakan untuk menyimpan file hasil coding, aplikasi web atau eksternal file agar dapat diakses oleh berbagai macam jenis perangkat (desktop dan mobile).

Cara Kerja Hosting adalah

Web Hosting juga erat kaitannya dengan domain fungsi domain tersebut berfungsi sebagai alamat file, sebagai contoh untuk mengakses sebuah file yang berisi dokumen html kita biasa menuliskan localhost:8158/index.html 

Penjelasan:

  • localhost:8158 adalah alamat tempat file disimpan.
  • index.html adalah nama file yang akan diakses.

Begitu pula dengan web hosting dalam menggunakan web hosting kita harus memiliki domain sebagai alamat file, jika kita menuliskan dupancode.my.id/materi/html1 maka yang terjadi:

Browser akan request ke server dengan domain dupancode.my.id/ lalu meminta file yang bernama materi/html1 jika file tersebut ada di penyimpanan dupancode.my.id/ maka akan direspon oleh server dan di krimkan kepada browser anda.

Berikut beberapa jenis layanan web hosting:

Shared Hosting

Layanan hosting jenis ini adalah layanan hosting yang digunakan oleh banyak user secara bersama-sama.

Kelebihan:

  • Harga lebih murah atau gratis.
  • Server dirawat sepenuhnya oleh penyedia layanan (full managed).
  • Fasilitas lengkap.
  • Dapat digunakan oleh banyak user.

Kekurangan:

  • Jika traffic pengunjung sedang ramai website akan lambat untuk diakses.
  • Menggunakan subdomain.
  • Mempunyai limit penyimpanan.

Layanan hosting gratis:

  • 000webhost
  • github
  • raplit

VPS Hosting

Virtual Private Server adalah sebuah layanan hosting yang bersifat milik pribadi dengan kata lain resourcenya hanya dapat digunakan oleh satu user.

Kelebihan:

  • Website tidak terpengaruh oleh traffic pengunjung website lain karena bersifat private server.
  • Memiliki akses untuk mengelola lebih dari satu akun hosting.
  • Dapat memilih sistem operasi.

Kekurangan:

  • Melakukan konfigurasi secara mandiri.
  • Perlu mempunyai pengalaman tentang server hosting.

Cloud Hosting

Layanan hosting jenis ini adalah gabungan antara shared hosting dan VPS hosting.

Kelebihan:

  • Resource yang diberikan oleh penyedia layanan hosting banyak.
  • Dapat berjalan dengan lancar dengan kondisi traffic pengunjung tinggi.
  • Server dirawat sepenuhnya oleh penyedia layanan (full managed).
  • Tidak perlu mempunyai pengalaman tentang server hosting.

Kekurangan:

  • Biaya layanan lebih maha dibandingkan dengan menggunakan VPS dan shared hosting.
