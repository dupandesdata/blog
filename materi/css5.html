Background

background: value;

Property ini digunakan untuk memberikan warna pada element html.

Property ini memiliki beberapa value sebagai berikut

Jenis Value
Color red, yellow, green
Hexadecimal #fff, #000, #00ff00
Red-green-blue rgba(108.58.63), rgba(0%.100%.50%)
Hue-saturation-lightness hsl(245.75%.50%)
Hue-saturation-lightness-alpha hsla(245.75%.50%.0.1)

Untuk penggunaannya seperti ini:

Source Code

<style>

.box {

width: 100px;

height: 100px;

background: #888;

}

</style>


<div class="box"></div>

background-image

background-image: url;

Property ini digunakan untuk membuat gambar menjadi background pada element html, value dari aproperty ini adalah url gambar.

Untuk penggunaannya seperti ini:

Source Code

body { background-image: image.jpg }

Property mempunyai kelemahan jika kita memasukkan gambar yang berukuran kecil maka akan diduplikat sebanyak ukuran element html yang diberikan gambar tersebut.

Penjelasan:

Element html memiliki ukuran 400x400px dan gambar yang kita sediakan ukuran 100px maka gambar tersebut akan diduplikat sebanyak 4buah secara otomatis. Cek disini

Background-repat

Property ini digunakan untuk melengkapi property background-image yang berfungsi untuk mengatur pengulangan gambar, baik itu secara horizontal dan vertikal property ini memiliki empat nilai sebagai berikut:

  1. no-repeat
  2. repeat
  3. repeat-x
  4. repeat-z

Penjelasan:

  1. no-repeat adalah perintah yang membuat gambar tidak melakukan pengulangan secara horizontal.
  2. repeat adalah perintah yang membuat gambar melakukan pengulangan secara horizontal dan vertikal.
  3. repeat-x adalah perintah yang membuat gambar melakukan pengulangan secara horizontal.
  4. repeat-z adalah perintah yang membuat gambar melakukan pengulangan secara vertikal.

Source Code

<style>

body {

background-image: /image.jpg;

background-repat: no-repat;

}

</style>

Bckground-position

Property ini digunakan untuk mengatur posisi gambar mirip dengan text-align yang berfungsi mengatur posisi element html sedangkan background-position digunakan untuk melengkapi property background-image.

background-position: value;

Ada beberapa value yang bisa kita gunakan:

Value Value
left top right top
left center right center
left bottom right bottom
center top 50% 50%
center center 50px 50px
center bottom initial

Untuk penggunaannya seperti ini:

Source Code

body {

background-image: url('gambar.png');

background-repeat: no-repeat;

background-position: right top;

}

Background-attachment

background-attachment: value;

Property ini digunakan untuk membuat background gambar ikut scrolling mouse atau tidak, property ini sering digunakan sebagai background fixed jika mouse di scroll gambar tersebut tidak mengikuti scroll mouse.

Property ini memiliki beberapa nilai:

Value Deskripsi
scroll background bergulir jika mouse dicroll
fixed background tidak bergulir atau posisi tetap jika mouse di scroll
local background akan bergulir sesuai dengan isi content
initial nilai standarnya/default

Background-size

Property ini digunakan sebagai pelengkap dari background-image yang berfungsi mengatur ukuran background image agar sesuai dengan kebutuhan.

background-size: value;

Berikut beberapa value yang bisa digunakan:

Value Deskripsi
px, em, mm, cm, dll Digunakan untuk lebar dan tinggi dari gambar background.
% Digunakan untuk lebar dan tinggi dari gambar background dengan ukuran persen.
auto Digunakan untuk menampilkan gambar background dengan ukuran asli dari gambar.
contain resize image, merubah ukuran gambar dan memastikan gambar terlihat sepenuhnya
cover membuat gambar background fullscreen.
initial setel value ke default.

Color

color: value;

Properti ini digunakan untuk mengatur warna teks element tertentu karena pada dasarnya semua element HTML memiliki warna teks hitam secara default, misalnya kita membuat sebuah button yang memiliki background hijau dan teks nya berwarna putih.

Berikut ini adalah value yang bisa kita gunakan untuk memberi warna teks:

Jenis Value
Color red, yellow, green
Hexadecimal #fff, #000, #00ff00

Source Code

<style>

.box {

width: 100px;

height: 50px;

background: #888;

color: red;

}

</style>
