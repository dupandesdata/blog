Form Input Validasi

Dupandes Milenium KD.

15 November 2022

Tag input adalah sebuah element kontrol interaktif yang digunakan untuk menerima data dari user. Element ini merupakan tag paling kompleks dari semua tag html karena tag input biasanya memiliki banyak sekali atribut yang berfungsi untuk memvalidasi data yang di masukkan oleh user.

Berikut adalah jenis-jenis type yang biasa digunakan:

  1. <input type="checkbox">
  2. <input type="submit">
  3. <input type="text">
  4. <input type="password">
  5. <input type="email">

Cek selengkapnya  

Berikut ini beberapa atribut validasi data yang bisa kita gunakan:

Atribut Minlength

minlegth="n"

Atribut ini digunakan untuk membatasi minimum karakter, misalnya user harus menuliskan nama lengkap tidak mungkin nama lengkap berisi satu karakter saja? Karena itu kita bisa set minimal karakter yang harus dituliskan oleh user.

Source Code

<form action=" " method="post">

  <label>Siapa nama kamu 

   <input type="text" minlength="5" />

  </label>

</form>

<style>

  input:invalid {

    color: white;

    background: red;

  }

</style>

Penjelasan:

Pada atribut minlength, n diisi dengan jumlah minimal karakter yang harus dituliskan pada input tersebut, source code diatas adalah membuat element input menjadi merah jika user menuliskan kurang dari 5 karakter.

Atribut Maxlength

maxlength="n"

Atribut ini digunakan untuk membatasi maximum karakter, misalnya user harus menuliskan nama lengkap yang berisi minimal 5 karakter dan maximal 25 karakter.

Source Code

<form action=" " method="post">

  <label>Siapa nama kamu 

   <input type="text" minlength="5" maxlength="25"/>

  </label>

</form>

<style>

  input:invalid {

    color: white;

    background: red;

  }

</style>

Penjelasan:

Pada atribut minlength n diisi dengan jumlah maximal karakter yang harus dituliskan pada input tersebut, source code diatas adalah membuat element input menjadi merah jika user menuliskan kurang dari 5 karakter dan lebih dari 25 karakter.

Atribut Required

required

Atribut ini digunakan agar user diwajibkan untuk mengisi data yang diperlukan sebelum dikirim.

Source Code

<form action=" " method="post">

  <label>Siapa nama kamu

   <input type="text" minlength="5" required/>

  </label>

  <input type="submit" value="Krim"/>

</form>

<style>

  input:invalid {

    color: white;

    background: red;

  }

</style>

Penjelasan:

Jika user belum menulis nama lengkap lalu mengklik tombol krim itu tidak akan bisa.

Atribut ini dapat digunakan pada input type: text, number, search, email, telp, url, checkbox, file, radio, dan date.

Atribut Min

min="n"

Atribut ini sama seperti minlength tetapi digunakan untuk input type number.

Source Code

<form action=" " method="post">

  <label>Masukan 2 digit angka

   <input type="number" min="2" required/>

  </label>

  <input type="submit" value="Krim"/>

</form>

<style>

  input:invalid {

    color: white;

    background: red;

  }

</style>

Penjelasan:

Pada atribut min, n diisi dengan jumlah minimal digit yang harus dituliskan pada input tersebut, source code diatas adalah membuat element input menjadi merah jika user menuliskan kurang dari 2 digit angka. 

Atribut Max

max="n"

Atribut ini sama seperti maxlength tetapi digunakan untuk input type number.

Source Code

<form action=" " method="post">

  <label>Masukan 2 - 5 digit angka

   <input type="number" min="2" max="5" required/>

  </label>

  <input type="submit" value="Krim"/>

</form>

<style>

  input:invalid {

    color: white;

    background: red;

  }

</style>

Penjelasan:

Pada atribut max, n diisi dengan jumlah maximal digit yang harus dituliskan pada input tersebut, source code diatas adalah membuat element input menjadi merah jika user menuliskan kurang dari 2 dan lebih dari 5 digit angka.

Atribut Placeholder

placeholder="keterangan"

Atribut ini digunakan untuk menampilkan petunjuk tentang penulisan data, misalnya sampel atau deskripsi singkat mengenai format yang ditetapkan).

Source Code

<form action=" " method="post">

  <label>Siapa nama kamu

   <input type="text" minlength="5" placeholder="Chandra Kirana" required/>

  </label>

  <input type="submit" value="Krim"/>

</form>

<style>

  input:invalid {

    color: white;

    background: red;

  }

</style>

Penjelasan:

Pada atribut ini, keterangan ditulis dengan format yang sudah diberikan, contoh penulisan nama lengkap kita sudah set minimal 5 karakter dan maximal 25 karakter kita bisa menggunakan placeholder untuk memberi tau kepada user penulisan nama yang benar.

Atribut Pattern

pattern="/modifier(s)/Expression"

adalah ekspresi reguler yang valueharus cocok dengan input agar nilai lulus validasi batasan

Atribut ini digunakan sebagai ekspresi reguler/format penulisan data yang harus dituliskan oleh user dengan format yang sudah ditetapkan, misalnya untuk menuliskan nama lengkap hanya boleh menggunakan huruf a-z saja.

Atribut ini dapat digunakan pada input type: text, number, search, email, telp, url

Berikut beberapa format yang kita bisa gunakan:

Modifier Keterangan
g Pencocokan global
i Pencocokan case-insensitive
m Pencocokan case-insensitivem Pencocokan multiline
Expression Keterangan
[abc] Menentukan karakter penulisan input hanya (A -Z)
[^abc] Menentukan karakter penulisan input tidak boleh menggunakan (A -Z)
[0-9] Menentukan karakter penulisan input type number (0 - 9)
[^0-9] Menentukan karakter penulisan input tidak boleh menggunakan (0 - 9)
(x|y) Menentukan salah satu alternatif yang ditentukan

Berikut ini format validasi input dengan regular expression.

<input pattern="regexp">

Metacharacter Keterangan
/d Menentukan angka
{n,} Menentukan jumlah karakter
Regexp Keterangan
(?=.*[A-Z])(?=.*[^0-9]).{5,} Memvalidasi nama
[a-z0-9.@-_]{5,} Memvalidasi email
(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,} Memvalidasi password
[+62]{1}[0-9]\d{13} Memvalidasi nomor telpon Indonesia
