Overflow

Dupandes Milenium KD.

12 Desember 2022

 Properti ini digunakan untuk mengontrol area konten/child element jika melebihi batas ukuran yang sudah ditetapkan, contoh sederhana ketika konten/child element melebihi width device maka secara otomatis browser akan menambahkan scrollbar hal ini bisa diatasi dengan overflow. Perinsip dasar overflow adalah memotong/menyembunyikan sebuah konten yang melebihi batas width/height.

overflow: visible | hidden | scroll | auto | inherit
Value Keterangan
Visible Ini adalah value default tidak berpengaruh apapun
Hidden  Digunakan untuk memotong konten/element yang melebihi batas ukuran yang sudah ditetapkan
Scroll Digunakan untuk membuat scrollbar sumbu x dan y
Auto Digunakan untuk membuat scrollbar jika konten/element sudah melebihi batas ukuran width/height yang ditetapkan
Syntax Keterangan
Overflow-X Digunakan untuk memotong sumbu x (horizontal)
Overflow-Y Digunakan untuk memotong sumbu y (vertikal)

Kita juga dapat menggunakan min dan max untuk memunculkan scrollbar, jadi ketika konten belum melebihi batas ukuran min dan max scrollbar tidak muncul.

Source Code

div {
  width: fit-content;
  height: 150px;
  max-height: 300px;
  overflow: auto;
}

Body overflow

Ini digunakan untuk membuat agar tidak terjadi scrollbar horizontal ketika ada sebuah element yang melebihi dari width body, untuk menggunakan nya kita harus menuliskan bersama dengan meta tag seperti ini:

Source Code

<head>
  <title>My Web</title>
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <style>
    body {
      overflow-x: hidden;
    }
  </style>
</head>
❛❛ Catatan:

Untuk penempatan meta tag harus dituliskan sebelum </head> .
