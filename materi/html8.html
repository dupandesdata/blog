Membuat Teks

 Untuk membuat sebuah teks kita dapat menggunakan semua jenis tag HTML namun dalam membuat teks terdapat tag-tag khusus untuk yang digunakan untuk membuat teks, ibaratkan seperti anda minum air dengan menggunakan selain gelas seperti mangkuk bisa si bisa tetapi bukan kegunaannya alangkah baiknya jika menggunakan gelas untuk meminum air. Sama seperti HTML ada tag-tag yang alangkah baiknya jika difungsikan dengan tepat seperti halnya gelas untuk meminum air bukan mangkuk, berikut ini adalah tag HTML yang tepat untuk membuat teks.

Tag <p>

Syntax HTML

<article>

  <p>Paragraf</p>

</article>

Tag ini digunakan untuk membuat paragraf tag <p> yang di bungkus dengan tag <article> sebaiknya tidak diberikan atribut apapun.

Source Code

<article>

  <p>Paragraf 1</p>

  <p>Paragraf 2</p>

  <p>Paragraf 3</p>

</article>

Paragraf 1

Paragraf 2

Paragraf 3

Dalam penggunaan tag <p> disarankan untuk tidak boleh menjadi parent tag utama dari sebuah content, tag <p> biasanya digunakan untuk menjadi parent tag <span> dan tag <a> tag lainnya yang digunakan untuk membuat teks, berikut adalah penggunaan tag <p> yang benar seperti berikut:

Source Code

<!-- penggunaan yang benar -->

<section>

  <article>

    <p>

      Saya belajar <a href="https://dupancode.my.id/materi/html5"> <span color="red">HTML</span></a>

    </p>

  </article>

</section>

   Saya belajar <a href="https://dupancode.my.id/materi/html5"> <span color="red">HTML</span></a>

</p>

Tag <span>

Syntax HTML

<span style="  " atribut=" ">Teks</span>

Tag <span> digunakan untuk memberikan styling embedded CSS contohnya membuat teks tertentu menjadi berwarna pada paragraf, tag ini juga bisa menambahkan semua jenis atribut dasar.

Source Code

<p>

    Saya belajar <span style="color: red">HTML</span>

</p>

Saya belajar HTML

Penjelasan:

Atribut style dapat berisi beberapa property dan value CSS.

Dalam penggunaan nya tag <span> disarankan tidak boleh menjadi parent tag lain, tag ini biasanya memiliki parent seperti berikut:

Source Code

<!-- penggunaan yang benar -->

<article>

        <p>

            Saya belajar <span color="red">HTML</span>

        </p>

</article>


<!-- penggunaan yang salah -->

<span>

    <span>Ini nama element</span>

        <span>Ini paragraf</span>

        <span>

            Saya belajar <span color="red">HTML</span>

        </span>

</span>

Tag <strong>

Syntax HTML

<strong>Teks</strong>

Tag ini digunakan untuk membuat teks tertentu menjadi tebal dan kuat untuk menandakan bahwa teks tersebut penting.

Source Code

<!-- penggunaan yang benar -->

<section>

  <article>

    <p>

      Saya belajar <strong>HTML</strong>

    </p>

  </article>

</section>
