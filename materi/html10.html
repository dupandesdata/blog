List Item

Daftar atau list adalah sebuah baris yang memiliki nomor atau simbol yang digunakan untuk memberi urutan dari sebuah daftar, didalam pemrograman HTML terbagi menjadi dua bagian yaitu: 

Ordered dan Unordered

Daftar Ordered 

Syntax HTML

<ol>

  <li>  </li>

<ol>

Dalam membuat daftar ordered atau daftar  berurut kita dapat menggunakan tag <ol> yang didalam nya terdapat tag <li> </li> (list item).

Source Code

<ol>

  <li>list item 1</li>

  <li>list item 2</li>

  <li>list item 3</li>

</ol>
  1. list item 1
  2. list item 2
  3. list item 3

Tag <ol> bisa menggunakan semua atribut dasar. Berikut atribut yang hanya dimiliki oleh tag <ol> .

Reversed

Syntax HTML

<ol reversed>

  <li>  </li>

<ol>

Atribut ini adalah atribut boolean digunakan untuk membuat urutan daftar menurun (9,8,7...), bukan menaik (1, 2, 3...).

Source Code

<ol reversed>

  <li>list item 1</li>

  <li>list item 2</li>

  <li>list item 3</li>

</ol>
  1. list item 1
  2. list item 2
  3. list item 3

Start

Syntax HTML

<ol start="  ">

  <li>  </li>

</ol>

Atribut ini digunakan untuk mnentukan nilai awal dari daftar pertama.

Source Code

<ol start="20">

  <li>list item 1</li>

  <li>list item 2</li>

  <li>list item 3</li>

</ol>
  1. list item 1
  2. list item 2
  3. list item 3

Syntax HTML

<ol type="  ">

  <li>  </li>

</ol>

Atribut ini digunakan untuk penanda dalam daftar beruruta (huruf atau angka), nilai ini selalu berupa bilangan bulat, meskipun jenis penomorannya adalah angka atau romawi. 

Nilai Atribut

Nilai Keterangan
1 Nilai Default (1, 2, 3, 4 ....)
a Nilai berdasarkan abjad huruf kecil (a, b, c, d ....)
A Nilai berdasarkan abjad huruf kapital (A, B, C, D ....)
i Angka Romawi keci (i, ii, iii, iv ....)
I Angka Romawi kapital (I, II, III, IV ....)

Daftar Unordered

Syntax HTML

<ul>

  <li>  </li>

<ul>

Tag ini digunakan untuk membuat daftar yang tidak berurutan (berpoin), tag ini juga bisa menggunakan semua atribut dasar.

Source Code

<ul>

  <li>list item 1</li>

  <li>list item 2</li>

  <li>list item 3</li>

<ul>
  • list item 1
  • list item 2
  • list item 3
