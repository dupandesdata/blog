Membuat Container Dasar

 Dalam membuat sebuah website terdiri dari beberapa container, perhatikan gambar berikut:

Header
<h1>
<nav>
Main
<aside>
<article>
<section>
<section>
<section>
<aside>
footer
<h6>

Kotak yang berwarna merah adalah container utama, container utama berfungsi sebagai wadah untuk menampung satu atau lebih element berikut ini adalah unsur-unsur yang dimiliki oleh container utama.

Margin Umum

Dengan menggunakan container utama kita tidak perlu memberikan margin element pada setiap element.

Font Umum

Ini mempermudah kita untuk memberi jenis font.

Position

Ini biasanya digunakan saat kita berkerja dengan menggunakan flexbox atau grid yang dimana kita tidak perlu menghitung jumlah margin untuk setiap element-element.

Dalam pembuatan container pada HTML 4 adalah menggunakan tag <div> dan di bedakan oleh class atau id seperti berikut:

wpbeginner.com

Hal ini tentunya akan sangat membingungkan karena container dengan element terbuat oleh tag yang sama, dengan adanya HTML 5 maka kita dapat membedakan antara container dan element yang berada didalam container tersebut.

Berikut beberapa tag yang digunakan untuk membuat container.

Tag <header>

Tag ini digunakan untuk membuat container header yang berisi:

  • <h1> (nama website)
  • <nav> (navigasi utama)
Header
<h1>
<nav>

Tag <main>

Tag ini digunakan untuk membuat container yang berisi beberapa container didalam nya:

  • <article> (content utama website)
  • <aside> (sidebar kanan atau kiri)
Main
<aside>
<article>
<section>
<section>
<section>
<aside>

Tag <article>

Tag ini digunakan untuk membuat artikel dan beberapa container didalam nya:

  • <h2> - <h5> (judul artikel atau subjudul)
  • <p> (content paragraf)
  • <section> (container menampilkan iklan)

Tag <footer>

  • Tag ini digunakan untuk membuat footer:
  • <h6> (copyright atau informasi lain tentang website)
