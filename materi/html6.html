Tag Heading

 Tag heading terdiri dari <h1> sampai ha <h6> tag ini digunakan untuk menuliskan judul, sub judul dan topik halaman, tag ini berhubungan dengan SEO (search engine optimization) karena search engine seperti Google, Yahoo dan lainnya akan mengindex suatu halaman berdasarkan judul halaman terlebih dahulu.

Syntax HTML

<h1> </h1>

<h2> </h2>

<h3> </h3>

<h4> </h4>

<h5> <h5>

<h6> </h6>

Output default 

ini adalah h1

ini adalah h2

ini adalah h3

ini adalah h4

ini adalah h5

ini adalah h6

Penjelasan:

Fungsi <h1> : tag ini digunakan untuk membuat nama website, untuk ukuran font secara default adalah 2em dan tag ini hanya ada satu pada halaman sebuah website.

Fungsi <h2> : tag ini digunakan untuk membuat sub-judul pada halaman website, untuk ukuran font secara default adalah 1.5em dan tag ini beberapa kali muncul pada halaman website.

Fungs <h3> : tag ini digunakan untuk membuat sub-topik pada halaman website, untuk ukuran font secara default adalah 1.3em dan tag ini beberapa kali muncul pada halaman website.

Fungsi <h4> : tag ini  digunakan untuk membuat judul “Halaman Terkait” atau juga digunakan untuk membuat judul widget seperti sidebar, untuk ukuran font secara default adalah 1.17em dan tag ini beberapa kali muncul pada halaman website dan tag ini beberapa kali muncul pada halaman website.

Fungsi <h5> : tag ini digunakan untuk membuat nama pemberi komentar, untuk ukuran font secara default adalah 0.83em.

Fungsi <h6> : tag ini digunakan untuk membuat nama footer seperti “All Rights Reserved”, untuk ukuran font secara default adalah 0.67em.

Tugas buatlah sebuah halaman website yang menguntungkan semua tak diatas.
