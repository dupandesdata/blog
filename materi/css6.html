Font Properti

Dupandes Milenium KD.

 Font property adalah sebuah styleling CSS yang digunakan untuk memberi gaya teks, mengatur ukuran font dan mengatur jenis font yang akan ditampilkan dalam browser. Agar terlihat lebih rapi dan menarik, seperti halnya Microsoft Word kita juga dapat memberikan ketebalan teks, mengatur teks size, jenis font.

Font Style

font-style: value;

Properti ini digunakan untuk menetapkan font family bagaimana font tersebut akan ditampilkan pada element html, apakah normal, miring, atau miring dari file.

Value Keterangan
normal Digunakan untuk menampilkan tulisan pada umumnya
italic Digunakan untuk membuat tulisan menjadi miring
oblique Digunakan untuk membuat tulisan lebih miring
oblique 10deg Digunakan menentukan kemiringan font 1-45deg

Font Size

font-size: value;

Property ini berfungsi untuk memberi ukuran pada tulisan, kita bisa merubah ukuran font pada elemen html.

Value Keterangan
px Pixels digunakan untuk memberi ukuran fix pada element/font.
pt Points digunakan untuk memberi ukuran font.
vh Viewport digunakan untuk memberi ukuran element/font yang ditampilkan sesuai layar.
rem Rem ukuran font relatif pada elemen root.
% Persen digunakan untuk memberikan ukuran element/font sesuai dengan lebar/tinggi element parent.

Font Family

font-family: value;

Properti ini digunakan untuk menentukan daftar font pada element HTML dari daftar tersebut terbagi menjadi dua yaitu:

  • Font prioritas adalah font utama dari sebuah element, font ini ditandai dengan menggunakan tanda petik dua.
  • Font alternatif adalah font-font yang di dukung oleh browser jika font prioritas tidak di dukung oleh browser.

Oleh sebab itu value font family dapat lebih dari satu dan dipisahkan dengan tanda koma untuk pada akhir jenis font, di sarankan untuk menggunakan satu font untuk halaman website ini mempengaruhi kecepatan sebuah website semakin banyak font yang digunakan semakin lama loading website.

Source Code

<style>

  @import url("https://fonts.googleapis.com/css2?family=Rubik+Distressed&display=swap");

  .teks {

       font-family: Arial, Helvetica, sans-serif;

}

   .teks2 {

       font-family: "Rubik Distressed", cursive, sans-serif;

}

</style>

<p class="teks">Ini teks Arial</p>

<p class="teks2">Ini teks Rubik Distressed</p>

Font Weight

font-weight: value;

Properti ini digunakan untuk mengatur ketebalan font, jenis value dalam property ini yaitu:

Value Keterangan
normal Tidak ada yang terjadi atau sama seperti 400
bold Digunakan untuk membuat teks tebal atau seperti 700
bolder Digunakan untuk membuat teks lebih tebal dari element parent (relatif)
number Number Digunakan untuk membuat teks tebal sesuai dengan nilai yang diberikan, nilai yang dapat digunakan antara 100 hingga 900
