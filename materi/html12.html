Tag Tabel

Dupandes Milenium KD.

11 November 2022

 Tabel digunakan untuk membuat sebuah data agar dapat dibaca dengan mudah, seperti halnya dengan menggunakan Excel table pada HTML terbuat dengan tiga unsur yaitu:

  • Baris
  • Kolom
  • Sel

Ada beberapa tag khusus untuk membuat tabel pada html seperti berikut:

Tag <table>

Digunakan untuk  mendefinisikan bahwa kita akan membuat tabel.

Tag <th>

kepanjangan dari th adalah table head yang digunakan untuk membuat judul pada tabel.

Tag <thead>

Digunakan sebagai pembungkus tabel bagian kepala.

Tag <tbody>

Digunakan sebagai pembungkus body dari tabel.

Tag <tr>

Kepanjangan dari tr adalah tabel row yang digunakan untuk membuat baris baru.

Tag <td>

Kepanjangan dari td adalah table data digunakan untuk membuat sel baru.

Struktur dasar tabel seperti ini:

Source Code

<table>

  <tbody>

    <thead>

      <th>NAMA</th>

      <th>USIA</th>

    </thead>

    <tr>

      <td>Dani</td>

      <td>20 tahun</td>

    </tr>

    <tr>

      <td>Ayu</td>

      <td>18 tahun</td>

    </tr>

    <tr>

      <td>Dita</td>

      <td>19 tahun</td>

    </tr>

  </tbody>

</table>
NAMA USIA
Dani 20 tahun
Ayu 18 tahun
Dita 19 tahun

Untuk mempercantik tampilan tabel kita bisa menggunakan atribut berikut ini:

Atribut Border

Atribut ini hanya terdapat satu pada satu tabel atribut border hanya di letakan pada tag <table> saja.

Syntax HTML

<table border="n">

...

<table>

Penjelasan:

n diisi dengan tingkat ketebalan garis semakin tinggi angka nya semakin besar pula garis nya.

NAMA USIA
Dani 20 tahun
Ayu 18 tahun
Dita 19 tahun

Atribut Rowspan

Atribut ini digunakan untu menggabungkan beberapa baris menjadi satu sell, biasanya digunakan pada tag <th> atau <td> .

Syntax HTML

<th rowspan="n"></th>

<td rowspan="n"></td>

Penjelasan

n diisi dengan jumlah baris yang akan digabungkan menjadi satu sell.


Atribut Colspan

Atribut ini digunakan untu menggabungkan beberapa kolom menjadi satu sell, biasanya digunakan pada tag <th> atau <td> .

Syntax HTML

<th colspan="n"></th>

<td colspan="n"></td>

Penjelasan

n diisi dengan jumlah kolom yang akan digabungkan menjadi satu sell.

tag ini juga bisa menambahkan semua jenis atribut dasar.
