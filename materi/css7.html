 Halaman website secara default (bawaan) jika tidak diberikan font family akan menggunakan font Times New Romon, kenapa font ini digunakan sebagai font standar dokumen HTML? Karena font tersebut di dukung oleh sistem operasi Windows, Linux, Mec dan Android. Didalam CSS ada yang disebut dengan property font-family yang digunakan untuk merubah sebuah font, perlu diperhatikan bahwa tidak semua sistem operasi mendukung font yang kita tentukan, oleh karena itu kita perlu menetapkan font family lebih dari satu.

font-family: calibri, arial, sans-serif;

Mengapa kita harus menentukan lebih dari satu font? Karena sistem operasi yang mengunjungi website berbeda-beda ketika user menggunakan sistem operasi Linux maka font calibri tidak ditemukan pada sistem operasi tersebut, karena didalam font family memiliki font arial jika user menggunakan Linux maka akan font yang digunakan adalah arial dan begitu seterusnya.

Sans-serif dan Serif

Pada dasarnya typography memiliki jenis font yaitu serif dan sans-serif, jenis font yang tidak memiliki kaki (strokes) pada ujungnya maka disebut dengan sans-serif, sedangkan jenis font serif memiliki kaki (strokes) pada ujungnya.

Sans-serif

Hello world 

Penggunaan

Font serif adalah jenis font yang memiliki garis horizontal pada setiap bagian ujung huruf, ini disebut dengan serif bracketed (counter stroke) font ini cocok digunakan untuk penulisan artikel karena tampak lebih mudah dibaca dan nilai readibility nya cukup baik.

Font sans serif adalah jenis font yang tidak memiliki garis horizontal pada setiap bagian ujung huruf, terlihat tampak solid dan lebih tegas. Font ini cocok digunakan untuk menulis judul halaman website.
