Dalam menyisipkan sebuah gambar pada halaman website tidak semua upload gambar pada sosial media seperti Facebook, gambar pada halaman website tersimpan dalam bentuk link dalam menampilkan gambar HTML meeyediakan tag khusus yaitu <img> .

Tag <img> adalah sebuah tag tunggal tidak memiliki tag penutup jadi tag pembuka sekaligus penutup.

Syntax HTML

<img/>

<img> <img/>

Tag <img> diibaratkan sebuah bingkai foto bagi file gambar kita bisa memasukkan gambar seperti .jpg, .gif dan lainnya dengan menggunakan atribut src .

Syntax HTML

<img src="link gambar"/>

Dalam membuat sebuah link gambar kita dapat menggunakan seperti google driver, media fire dan lain sebagainya, namun saya suka menggunakan blogger.com sebagai penyimpan gambar karena sudah diberikan Atribut pendukung.

Berikut beberapa atribut yang ada dalam Tag <img> .

Atribut height

Atribut ini digunakan untuk memberi tinggi pada gambar.

Syntax HTML

<img src="gambar.jpg" height="n"/>

Penjelasan:

n diisi dengan ketinggian gambar biasanya dalam bentuk px.

Atribut width

Atribut ini digunakan untuk memberi lebar pada gambar.

Syntax HTML

<img src="gambar.jpg" height="150px" width="n">

Penjelasan:

n diisi dengan lebar gambar biasanya dalam bentuk px.

Source Code

<img src="gambar.jpg" height="150px" width="150px"/>

Atribut alt

Atribut ini digunakan untuk membuat informasi jika gambar itu mengalami masalah sehingga tidak muncul pada halaman website.

Syntax HTML

<img alt="informasi gambar"/>

Source Code

<img src="gambar.jpg" height="150px" width="150px" alt="ini adalah gambar gunung"/>
