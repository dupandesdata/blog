Element Form

Dupandes Milenium KD.

14 November 2022

 Form digunakan sebagai interaksi user dengan sebuah sistem dengan cara mengirimkan data atau informasi yang diperlukan oleh sistem agar dapat berjalan, data yang diperoleh akan di proses disisi server kemudian akan dikembalikan kepada user hasil pemrosesan tersebut. 

Contoh sederhana penggunaan form adalah form login agar dapat berjalan form login membutuhkan data Username dan Password, data tersebut akan dikirim kepada server kemudian akan diproses untuk melakukan pengecekan ke database lalu akan dikembalikan kepada user hasil dari pemrosesan tersebut.

Berikut ini adalah beberapa bahasa pemrograman yang digunakan untuk memproses data yang diperoleh dari form:

  • PHP 
  • ASP
  • JSP
  • Python
  • Ruby
  • Dan lain sebagainya 

Untuk membuat form menggunakan tag berikut:

<form>...</form>

Terdapat dua jenis atribut form untuk mengirim data yaitu:

Action

method

Action

Atribut ini digunakan untuk menuliskan alamat file yang akan memeroses data contoh login.php maka data yang diperoleh akan di proses didalam file login.php

Source Code

<form action="login.php">

</form>

Method

Atribut ini digunakan berfungsi sebagai metode pengiriman data, data tersebut dikirimkan dengan cara URL atau dalam bentuk transaksi posting HTTP. Terdapat dua jenis method yaitu: 

method="get"

method="post"

method="get" berfungsi untuk:

  • Mengirimkan data dalam bentuk URL.
  • Panjang max dari URL adalah 3000 karakter.
  • Method get sangat sensitif terhadap kebocoran data karena data yang dikirim dalam bentuk URL sehingga dapat terlihat.

method="post" berfungsi untuk:

  • Mengirimkan data dalam bentuk permainan http request sehingga tidak dapat terlihat.
  • Tidak ada pembatasan jumlah karakter.
  • Tidak dapat di-bookmark.

Source Code

<form action="script/URL" method="get/post">

  .......

</form>

HTML Input

Ini adalah element yang wajib ada didalam struktur form element ini digunakan untuk interaksi antara program dengan user, sehingga user dapat memasukan data yang dibutuhkan oleh program agar program tersebut bisa berjalan, macam-macam type element input.

Berikut ini adalah beberapa element pendukung <form>

Fieldset

<fieldset>...</fieldset>

Element ini digunakan untuk mengelompokkan logic input.

Legend

<legend>...</legend>

Element ini digunakan untuk membuat  informasi dari kelompok kontrol input.

Button

<input type="submit"/>

Element ini digunakan untuk mengirim data yang sudah diberikan oleh user kemudian akan diproses oleh program tertentu dan akan di kembalikan ke user hasil dari pemrosesan data tersebut.
