Z-index

Dupandes Milenium KD.

8 Disember 2022

 Properti CSS yang digunakan untuk menentukan tumpukan element, dimana kita bisa memanipulasi element nilai z-index lebih besar akan menutupi element dengan nilai z-index yang lebih kecil.

Pada dasarnya semua element HTML memiliki display default inline atau black mereka tidak bisa di tumpuk, ada kalanya kita harus menumpuk element kita harus menggunakan properti position.

z-index: value;

Value yang bisa digunakan yaitu bilangan positif atau negatif yaitu dari 0 - tak terhingga.

Source Code

<style>
  .container {
    height: 100px;
    width: 250px;
    display: flex;
    position: relative;
    margin-top: 4px;
  }
  .container div {
    height: 50px;
    width: 80px;
    padding: 0 5px;
    left: 0;
    font-size: 9pt;
    position: absolute;
  }
  .container div:nth-child(1) {
    background: lightcoral;
  }
  .container div:nth-child(2) {
    top: 15px;
    left: 6px;
    background: lightseagreen;
  }
  .container div:nth-child(3) {
    top: 30px;
    left: 11px;
    background: rgb(167, 81, 201);
  }
  .container div:nth-child(4) {
    top: 45px;
    left: 16px;
    background: lawngreen;
  }
</style>

<section class="container">
  <div>element1</div>
  <div>element2</div>
  <div>element3</div>
  <div>element4</div>
</section>

❛❛ Catatan:

Jika element ditumpuk tanpa z-index maka element terakhir akan menepati paling depan.

Dengan menggunakan z-index kita dapat menampilkan element yang berada di urutan paling belakang ke depan

Source Code

 .container div:nth-child(1) {
  background: lightcoral;
  z-index: 1;
}
