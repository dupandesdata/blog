Pengertian Selector Dasar

Selector pada CSS digunakan untuk menampung property dan value styling agar dapat dipanggil pada atribut html.

Selector All

* {  }

Selector ini digunakan untuk memberi styling pada semua tag html biasa digunakan untuk me-reset border box element.

Source Code 

* {

    margin: 0;

    padding: 0;

}

Penjelasan:

Kode CSS diatas berfungsi untuk me-reset margin dan padding untuk semua element html, ini berguna supaya kita memiliki kontrol penuh terhadap tag-tag HTML.

Class Selector

.namaClas { }

Class Selector digunakan sebagai penampung style tertent, class selector harus diawali dengan tanda (titik) di depan nama selector.

Source Code

.myName {

    color: blue;

}

Penjelasan:

Nama class tidak boleh menggunakan spasi hanya boleh menggunakan tanda hubung - contoh:

Source Code 

.my-name {

     color: yellow;

}

Untuk menggunakan class tersebut harus dilakukan pemanggilan dengan atribut class="  "

Source Code 

<p class="myName">Dupandes</p>

<p class="my-name">Milenium</p>

Dupandes

Milenium

Walaupun memiliki nama selector yang sama tapi memiliki styling yang berbeda, kenapa bisa begitu? Karena nama class selector yang pertama tidak diberi tanda hubung (myName) dan nama class selector yang kedua diberi tanda hubung (my-name). Sesungguhnya kedua class tersebut adalah class yang berbeda walaupun memiliki kemiripan nama.

Class tersebut bisa kita gunakan lebih dari satu tag.

Source Code 

<p class="myName">Dupandes</p>

<p class="my-name">Milenium</p>

<p class="myName">Dani</p>

<p class="my-name">Arifin</p>

Dupandes

Milenium

Dani

Arifin

ID Selector

Sama seperti class selector cuma dalam penulisan id selector menggunakan tanda # 

Syntax Selector

#namaid { 

}

Source Code 

#prof {

     color: green;

}

Untuk memanggil id selector tersebut dengan menggunakan atribut id=" " pada tag html yang dituju:

Syntax Selector

<p id="prof">Dupandes</p>

Dupandes

Penjelasan:

Jika menggunakan id selector hanya bisa di panggil oleh satu tag saja karena bersifat unik yang hanya 1 didalam dokumen html.

Disarankan jika menggunakan id selector nama id menggunakan dua kata maka kata kedua diawali dengan kapital #myProf karena nanti nya terhubung dengan javascript.

Tag Selector

tag {  }

Selector jenis ini digunakan untuk memberi styling pada tag tertentu:

Source Code

}

Jika menggunakan element selector maka semua element yang menggunakan tag tersebut akan terkena style.

<p>paragraf 4</p>

Grouping Selector 

tag, tag , tag { }

Selector jenis ini digunakan untuk memberi styling pada lebih dari satu element dengan styling yang sama.

Source Code CSS

p, div, a {

  color: green;

}

Source Code HTML

<div>ini element</div>

<p>ini paragraf</p>

<a href="">ini link</a>

Penjelasan:

Kita juga dapat memberikan style multiple selector dengan menggunakan tanda (koma) , di akhir selector tersebut.

ini element

ini paragraf

ini link

Contoh styling multiple

Source Code

p, .link, section .judul, div p:first-child, section p: nth-child(4) { }
