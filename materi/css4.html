Properti Display Default

Dupandes Milenium KD.

18 November 2022

Display adalah salah satu properti CSS yang digunakan untuk mengontrol tata letak setiap element HTML bagaimana element tersebut ditampilkan pada browser,  setiap element HTML memiliki display default yang berbeda oleh karena itu setiap element HTML memiliki perilaku tampilan yang berbeda.

Display Block

display: block;

Display jenis ini digunakan untuk menampilkan element dimulai dengan baris baru, jika element HTML memiliki display ini ia tidak akan bisa berjejer ke samping sebagai tag <p>, <div>, <section> dan lain-lain. Element yang memiliki display ini dapat kita ubah/mengatur height dan width, hampir semua element HTML memiliki display default block.

Display Inline

display: inline;

Display jenis ini digunakan untuk menampilkan element tidak dimulai dengan baris baru, element yang memiliki display ini akan berjejer ke samping seperti tag <a>, <span>, <b> dan lain-lain. Kita tidak dapat mengubah/mengatur height dan width supaya kita bisa mengatur lebar dan tinggi nya kita bisa merubah display menjadi inline-block.

Display Inline-block

Display jenis ini adalah gabungan antara inline dan block yang digunakan untuk mengatur lebar dan tinggi element yang memiliki display default inline, lalu bagaimana cara untuk merubah display menjadi inline-block? Kita cukup menuliskan:

display: Inline-block

<span> adalah tag HTML yang memiliki display default inline ciri-ciri element yang berdisplay Inline ia akan berjejer ke samping dan tidak dapat diatur lebar dan tinggi nya.

Source Code

<style>

 span {

  width: 150px;

  height: 150px;

 background: green;

}

</style>

<span>element 1</span>

<span>element 2</span>
element 1 element 2

Penjelasan 

Tidak berpengaruh apapun karena kita belum mengubahnya menjadi inline-block.

Source Code

<style>

 span {

  display: Inline-block;

  width: 150px;

  height: 150px;

 background: green;

}

</style>

<span>element 1</span>

<span>element 2</span>
element 1 element 2

Kita juga dapat mengganti suatu tag menjadi display block atau Inline.

Source Code

<style>

 span {

 color: green;

}

 div {

 color: red;

}

</style>

<span>element span</span>

<span>element span</span>

<div>element div</div>

<div>element div</div>

Output secara default:

element span element span
element div
element div

Kita akan merubah display <div> menjadi inline dan <span> menjadi block, oh iya cara ini dapat dilakukan pada semua tag HTML.

Source Code

<style>

 span {

 display: block;

 color: green;

}

 div {

 display: inline;

 color: red;

}

</style>

<span>element span</span>

<span>element span</span>

<div>element div</div>

<div>element div</div>
element span
element span
element div element div

Display None

display: none;

Display jenis ini digunakan untuk menghilangkan element tertentu biasanya berguna saat kita membuat responsif, jika pada layar maksimal 700px element tersebut ditampilkan dan jika lebar layar minimal 900px maka element tersebut tidak ditampilkan.
