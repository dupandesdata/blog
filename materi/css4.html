Property Margin CSS

 Apa itu property margin:

Property margin adalah sebuah perintah untuk memberikan ruang antara satu elemen dengan elemen yang lainnya. Jarak yang di hasilkan oleh property ini adalah jarak bagian luar elemen, berikut ini adalah cara penulisan margin.

Syntax Property 

h1 { margin: 0 0 0 0; } 

Penjelasan:

  • Angka pertama adalah jarak bagian atas (top)
  • Angka kedua adalah jarak bagian kiri (left)
  • Angka ketiga adalah jarak bagian bawah (bottom)
  • Angka keempat adalah jarak bagian kanan (right) 

Saya ingin membuat sebuah elemen yang jarak top 30px left 50px bottom 20px right 10px.

Maka saya akan tulis seperti ini:

Syntax CSS

selector { 

margin: 30px 50px 20px 10px 

}

Jika kita mau semua jarak margin di setiap sisi nya sama kita hanya menuliskan satu angka yaitu margin: nolai; Ini akan membuat jarak top, left, bottom & right menjadi sama. 

Syntax CSS 

selector { 

margin: 40px;

}

Output:

Penjelasan:

Garis tepi kita anggap saja elemen lain.

Pada materi pelajaran Property dan Value CSS kita sudah mempelajari ada empat property pendukung yaitu: 

  • top
  • bottom
  • right
  • left

Kita bisa menggunakan property tersebut untuk mendukung property margin, dengan cara memberi tanda -  sebagai berikut:

Syntax Property

selector {

margin-top: value;

margin-left: value;

margin-bottom: value;

margin-right: value;

}

Penjelasan:

  • margin-top (hanya jarak atas)
  • margin-left (hanya jarak kiri)
  • margin-bottom (hanya jarak bawah)
  • margin-right (hanya jarak kanan)

Contoh:

Syntax CSS 

selector { 

margin-top: 20px;

}

Maka elemen yang diberi margin tersebut memiliki jarak bagian atas sebesar 20px.
