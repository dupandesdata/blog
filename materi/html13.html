Element Input

Dupandes Milenium KD.

14 November 2022

Element input adalah element berfungsi untuk jembatan yang menghubungkan antara sebuah program/server dengan user dalam interaksi menggunakan data, element ini memiliki banyak sekali atribut yang menjadikan element ini sangat kompleks dari element-element HTML lainnya.

Dalam penggunaan nya element ini terbagi dalam beberapa type seperti berikut:

<input/>

Type Text

Type ini digunakan untuk menuliskan huruf, angka dan simbol.

Source Code

<input type="text"/>

Type Number 

Type ini digunakan untuk menuliskan hanya angka saja.

Source Code

<input type="number"/>

Type Password

Type ini digunakan untuk menuliskan kata sandi/password agar tidak terlihat (hidden teks).

Source Code

<input type="password"/>

Type Email

Type ini digunakan untuk menuliskan hanya alamat email saja.

Source Code

<input type="email"/>

Type URL

Type ini digunakan untuk menuliskan hanya alamat website saja.

Source Code

<input type="url"/>

Type Tel

Type ini digunakan untuk menuliskan hanya nomor telepon saja.

Source Code

<input type="tel"/>

Type Search

Type ini digunakan untuk melakukan searching data misalnya mencari artikel.

Source Code

<input type="search"/>

Type File

Type ini digunakan untuk meng-upload file/foto yang diambil dari ponsel/komputer.

 Source Code

<input type="file"/>

Type Radio

Type ini digunakan untuk memilih salah satu jawaban dari pertanyaan yang diajukan dalam grup radio.

Source Code

<p>Beberapa usia anda</p>

<input type="radio"/> 10 -20 tahun

<input type="radio"/> 20 -30 tahun

<input type="radio"/> 30 -50 tahun

Type Checkbox

Type ini digunakan untuk memilih satu atau lebih dari pertanyaan yang diajukan dalam grup checkbox.

Source Code

<p>Apa hobi anda</p>

<input type="checkbox"/> Main game 

<input type="checkbox"/> Olahraga 

<input type="checkbox"/> Membaca

Type Range

Type ini digunakan untuk mendefinisikan nilai misalnya nilai review, nilai secara default adalah 1 - 100.

Source Code

<p>Berapa nilai dari makanan ini?</p>

<input type="range"/>

Type Button

Type ini digunakan untuk membuat tombol.

Source Code

<input type="button" value="Tombol"/>

Type Submit

Type ini digunakan untuk mengirim data yang sudah ditulis dalam input type text, email, number dan lain-lain biasanya type ini digunakan dalam form.

Source Code

<form>

<p>Tulis nama anda</p>

<input type="text"/>

<p>tulis umur anda</p>

<input type="number"/>

<input type="submit" value="Krim"/>

</form>

Type Reset

Type ini digunakan untuk me-reset data yang sudah ditulis dalam input type text, email, number dan lain-lain sebelum dikirim biasanya type ini digunakan dalam form.

Source Code

<form>

<p>Tulis nama anda</p>

<input type="text"/>

<p>tulis umur anda</p>

<input type="number"/>

<input type="reset" value="Reset"/>

<input type="submit" value="Krim"/>

</form>

Atribut Size

<input type=" " size="n"/>

Atribut ini digunakan untuk menentukan lebar dari element <input>, n diisi dengan jumlah lebar dari element <input> .

Source Code

<input type=" " size="20"/>

Tag Label

<label>...</label>

Element ini digunakan untuk menuliskan keterangan dari suatu element <input> dengan menggunakan tag <label>, jika kita klik bagian keterangan maka akan diberi tau dimana element <input> yang sesuai dengan keterangan tersebut.

Source Code 

<!-- cara yang pertama -->

<label>Masukan nama anda

 <input type="text" size="10"/>

</label>

<label>Masukan usia anda

 <input type="number" size="10"/>

</label>


<!-- cara yang kedua -->

<label for="nama">Masukan nama anda</label>

<input type="text" size="10" id="nama"/>

<label for="usia">Masukan usia anda</label>

<input type="number" size="10" id="usia"/>
