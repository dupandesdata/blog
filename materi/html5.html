Jenis Atribut Dasar HTML

Atribut dasar adalah atribut yang bisa digunakan oleh semua tag HTML, atribut ini digunakan untuk memanggil style CSS atau script kedalam element tertentu.

Attribute Description
class="  " Memanggil satu atau lebih nama selector CSS untuk suatu elemen (Internal/Eksternal CSS )
hidden Menentukan pada suatu elemen untuk tidak menampilkan pada browser 
id="  " Memanggil id selector CSS atau JavaScript yang bersifat unik 
style="  " Membuat Embedded CSS pada element tertentu
tabindex Menentukan urutan tabbing element tertentu 
title
 Membuat informasi tambahan tentang suatu element saat di hover 

Class Selector

Syntax HTML 

<element class="nama selector">content</element>

Dengan menggunakan atribut ini kita dapat memanggil selector CSS tertentu.

Source code

<style>

  .ex {

    color: red;

  }

</style>

<p class="ex">ini adalah contoh memanggil 1 class selector</p>

ini adalah contoh memanggil 1 class selector

Penjelasan:

Untuk menuliskan nama class selector harus didalam tanda "  " .

Kita juga bisa memanggil dua atau lebih class selector dalam satu element.

Source Code

<style>

  .ex {

    color: red;

  }

  .ex2 {

    font-size: 20px;

  }

</style>

<p class="ex ex2">ini adalah contoh memanggil 2 class selector</p>

ini adalah contoh memanggil 2 class selector

Penjelasan:

Dalam pemanggilan lebih dari satu nama class harus di pisahkan oleh spasi dan tetap berada didalam tanda "  " .

Hidden

Syntax HTML

<element hidden>content</element>

Atribut ini digunakan untuk menyembunyikan sesuatu element agar tidak ditampilkan pada browser tetapi jika kita cek kedalam source code terdapat element tersebut.

Source Code 

<p>ini element 1</p>

<p hidden>ini element 2</p>

Id Selector

Syntax HTML

<element id="nama selector">content</element>

Atribut ini digunakan untuk memanggil id yang berbeda di CSS atau JavaScript biasa di tulis dengan menggunakan # pada CSS, tidak seperti halnya dengan menggunakan atribut class selector atribut id hanya bisa diisi dengan satu id.

Source code

<style>

  #ex {

    color: rgb(0, 81, 255);

  }

</style>

<p id="ex">ini adalah contoh memanggil id selector</p>

ini adalah contoh memanggil id selector

Style

Syntax HTML

<element style="isi styling embedded CSS">content</element>

Atribut ini digunakan untuk membuat Embedded CSS pada element tertentu kita dapat menulis selayaknya menuliskan css tetapi tidak menggunakan selector, jika menggunakan style kita harus menutup tag tersebut agar tidak mempengaruhi element lainnya.

Source Code

<p style="color: rgb(0, 255, 34);">ini adalah contoh penggunaan atribut style</p>

ini adalah contoh penggunaan atribut style

Tabindex

Syntax HTML

<element tab="nomor">content</element>

Atribut ini digunakan untuk membuat urutan tab elemen, tabindex digunakan untuk memvalidasi di setiap elemen namun, tidak selalu berguna. (ketika tombol "tab" digunakan untuk navigasi).

Source Code

<p tabindex="1">ini element 1</p>

<p tabindex="2">ini element 2</p>

<p tabindex="3">ini element 3</p>

ini element 2

ini element 3

Title

Syntax HTML

<element title="informasi">content</element>

Atribut ini digunakan untuk membuat informasi tentang suatu element tertentu saat di hover oleh mouse.

Source Code

<p title="ini element 1">arahkan kursor ke sini</p>

arahkan kursor ke sini
