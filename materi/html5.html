Jenis Atribut HTML

1 Atribut

class="..." id="..." atribut ini kita bisa gunakan untuk memanggil style CSS dan perintah javascript, bersifat universal jadi semua tag html bisa menggunakan atribut ini.

2. Atribut 

href="..." atribut ini untuk memberi link ke halaman lain dan memanggil CSS eksternal, atribut ini tidak bisa digunakan untuk semua jenis tag html atribut ini digunakan pada tag <a> dan tag <link>

3. Atribut

src="..." atribut ini untuk memanggil link gambar dan link javascript eksternal, sama seperti halnya dengan href atribut ini tidak bisa digunakan pada semua tag html hanya dapat digunakan untuk tag <img> dan <script>

Cara membuat text ketika kita klik menuju halaman lain:

Syntax HTML 

<a href="https://dupancode.my.id">klik disini</a>

  1. Tag <a> adalah tag pembuka.
  2. Atribut harus berada di <a dan sebelum >
  3. Pada antara > dan </a> itu adalah tempat dimana kita menuliskan tulis dan elemen yang akan kita buat menjadi bisa di klik.
  4. </a> adalah tag penutup.

Syntax HTML 

<a href="..." class="..." id="...">elemen atau konten</a>

  • href untuk memberi link.
  • class untuk memberi style.
  • id untuk memanggil javascript.
