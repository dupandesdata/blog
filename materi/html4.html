Struktur Dokumen HTML Dasar

Pengertian

 HTML HTML merupakan singkatan dari (Hyper Text Markup Language) adalah bahasa pemrograman web terdiri dari beberapa simbol-simbol atau tag-tag yang ditulis dalam file index.html untuk ditampilkan pada halaman web browser.

Dalam membuat dokumen halaman website selalu diawali oleh tag <html> dan ditutup oleh tag </html> kita bisa menggunakan notepad dan menyimpan dalam bentuk ekstensi .html, ekstensi tersebut menunjukkan bahwa dokumen yang kita tulis berformat HTML untuk mempermudah proses penulisan tag-tag html kita bisa menggunakan Text Editor berikut ini beberapa text editor yang bisa kita gunakan:

  • Visual studio code
  • Raplit

Tag HTML

Tag html terdiri dari tag pembuka dan tak penutup sebagai berikut:

Tag Pembuka

<html>

Tag Penutup

</html>

Berikut beberapa tag yang harus dihafalkan:

Syntax HTML 

<html> </html>

<head> </head>

<title> </title>

<body> </body>

<main> </main>

<h1> </h1>

<h2> </h2>

<h3> </h3>

<h4> </h4>

<h5> <h5>

<h6> </h6>

<div> </div>

<section> </section>

<p> </p>

<footer> </footer>

Dokumen HTML

Dalam membuat dokumen html kita butuh beberapa tag yang wajib ditulis sebagai berikut:

Syntax HTML Dasar 

<html>

<head>

    <meta charset="UTF-8">

    <title>Judul halaman</title>

</head>

<body>

</body>

</html>

Penjelasan:

Tag <html> (HyperText Markup Language) berguna untuk membungkus semua element html, jadi ketika kita menuliskan element-element html harus berada di antara tag <html> dan </html>.

Tag <head> digunakan untuk membungkus tag meta, tag title, css dan script.

Syntax HTML

<head>

    <meta charset="UTF-8">

    <title>Judul halaman</title>


    <style>

        kode css

    </style>


    <script>

        kode javascript 

    </script>

</head>

Tag <meta> digunakan untuk memberitahukan kepada browser bagaimana dokumen html tersebut akan ditampilkan. 

Tag <body> digunakan untuk membungkus element yang akan di tampilkan pada browser, namun ada beberapa script yang harus ditulis diantara <body> dan </body>.

Syntax HTML 

<body>

    <h1>Judul Artikel</h1>

    <p>Paragraf 1</p>

    <p>Paragraf 2</p>

    <img src="" alt="">

    <p>Paragraf 3</p>

    

    <section>

        <div>Element 1</div>

        <div>Element 2</div>

    </section>

    

    <script>

        kode javascript 

    </script>

    

    <footer>Copyright 2020</footer>

</body>

Untuk membuat dokumen html yang baik kita harus memenuhi standar World Wide Web Consortium (W3C). Agar dokumen html yang kita buat dapat mendukung HTML 5 maka harus diawali dengan <!DOCTYPE html> (Document Type Declaration) yang berfungsi untuk menyatakan kepada browser bahwa dokumen yang kita tulis adalah tipe html yang sudah mendukung HTML 5.

Syntax HTML 

<!DOCTYPE html>

<html>

<head>

    <meta charset="UTF-8">

    <title>Judul halaman</title>


    <style>

        kode css

    </style>

    

    <script>

        kode javascript 

    </script>

</head>

<body>

    <h1>Judul Artikel</h1>

    <p>Paragraf 1</p>

    <p>Paragraf 2</p>

    <img src="" alt="">

    <p>Paragraf 3</p>

    

    <section>

        <div>Element 1</div>

        <div>Element 2</div>

    </section>

    

    <script>

        kode javascript 

    </script>

    

    <footer>Copyright 2020</footer>

</body>

</html>
