Setelah Mosaic Browser sukses menjadi browser populer dimasanya dua orang peneliti yang membuat Mosaic Browser mendirikan perusahaan yang diberi nama Netscape Communicator, perusahaan ini membuat sebuah browser yang diberi nama Netscape Navigator browser ini dibuat dengan fitur-fitur yang lain baik daripada Mosaic Browser seperti menambahkan seperti tag blink, tag formating, scripting dll tentu saja dengan tidak mempedulikan spesifikasi dari Tim Berners-Lee.

Tidak hanya itu saja ia juga ingin membuat Netscape Navigator lebih interaktif lagi seperti memberikan event saat user menekan tombol karena itu Netscape Communicator merekrut seseorang yang bernama Branden Eich, ia adalah seorang ahli sistem engineer setelah bergabung dengan Netscape Communicator ia di minta untuk membuat script seperti Java Script tersebut diberi nama mocha, ia membuat bahasa pemrograman yang menghubungkan bahasa pemrograman:

  1. Java
  2. Scheme
  3. Self

Karena waktu yang berdekatan dengan keluarnya Netscape Navigator 2.0 script mocha hanya dibuat dalam waktu 10hari, pada tahun 1995 mocha berganti nama menjadi LiveScript yang di implementasikan pada Netscape Navigator 2.0 dan mulai dikenalkan kepada publik.

Netscape Navigator 2.0 mendapat banyak kritik dari para pengguna browser tersebut merasa kurang aman karena dalam proses pembuatan Netscape Navigator 2.0 terkesan terburu-buru, oleh sebab itu Netscape Communicator bekerja sama dengan Sun. Sun Microsystems, Ins (Sun)adalah developer dari Java dengan adanya kerja sama tersebut developer Java berniat mengimplementasikan script Java kedalam browser Netscape Navigator 2.0 namun perusahaan Netscape Communicator tidak setuju karena LiveScript lebih teroktimasi pada web browser. Jadi diputuskan pada tanggal 4 Desember 1995 untuk mengganti nama LiveScript menjadi JavaScript supaya orang-orang mengira bahwa bahasa pemrograman Java yang digunakan pada web browser namun tetap menggunakan bahasa pemrograman memakai bahasa LiveScript, tetapi lisensi JavaScript diberikan kepada Sun dan hanya boleh digunakan pada Netscape Navigator 2.0

Persaingan Perusahaan

Dalam waktu yang bersamaan Microsoft meluncurkan Internet Explorer 3.0 mereka melakukan reverse engineering pada JavaScript yang diberi nama dengan JScript yang digunakan untuk mereka sendiri, perusahaan Netscape lalu mengajukan standarisasi kepada World Wide Web Consortium (W3C) untuk antisipasi supaya tidak di reverse engineering  seperti yang dilakukan oleh Microsoft namun, tidak diterima mungkin karena Netscape sudah melanggar peraturan Tim Berners-Lee.

Standarisasi JavaScript

Setelah gagal mendapatkan standarisasi dari W3c Netscape kemudian mengajukan standarisasi kepada European Computer Manufacturers Association (ECMA) dan pada tahun 1997 dibuatkan standar bahasa pemrograman JavaScript yang diberi nama ECMA-Script jadi semua browser sudah bisa menggunakan JavaScript. Kenapa penamaan standarisasi dengan nama ECMA-Script? Karena lisensi JavaScript dipegang oleh Sun.

Banyak yang mengira JavaScript, JScript dan ECMA-Script adalah bahasa pemrograman yang berbeda tetapi ketiga bahasa pemrograman tersebut adalah satu bahasa pemrograman.

Perkembangan Standarisasi JavaScript

  • Standar 1997, ES1 (ECMA-Script versi 1)
  • Standar 1998, ES2 (ECMA-Script versi 2)
  • Standar 1999, ES3 - Launch (ECMA-Script versi 3) Semua browser sudah support JavaScript.
  • Standar 2005, AJAX
  • Standar 2008, ES4 GAGAL (ECMA-Script versi 4) Standar ini gagal karena terdapat ketidak kompakan browser dan standar di buat ulang dengan nama ES Harmoni.
  • Standar 2009, ES5 (ECMA-Script versi 5) Pada saat munculnya ES web browser semakin interaktif dan mulai bermunculan JavaScript Libraries and Frameworks.

Standar yang digunakan saat ini adalah ES5.
