Type data JavaScript

 Di dalam sebuah sistem operasi komputer itu terdiri dari serangkaian bit, bit merupakan singkatan dari Binary Digit (Digit Biner) yang hanya memiliki dua angka yaitu 0 & 1. Angka ini dihasilkan oleh arus listrik angka 0 terjadi ketika komputer itu keadaan tidak dialiri arus listrik dan angka 1 terjadi ketika komputer dialiri arus listrik.

Jika didalam bilangan desimal memiliki satuan yaitu 100 | 10 | 1 dst, maka jika kita ingin mendapatkan angka 31 kita dapat mempresentasikan sebagai berikut :

Pembilang    |  3  |  1  | 

Penyebut      | 10 |  1  | = 31

Untuk menghasilkan bilangan 31 komputer mempresentasikan sebagai berikut : 11010

| 1 | 1 | 0 | 1 | 0 |

| 16 | 8 | 4 | 2 | 1 | = 16+8+4+2+1= 31

Setiap kali bertambah kekiri maka penyebut dikalikan 2 dari penyebut sebelumnya.

Xontoh 11101 penyelesaiannya:

|  1 | 1 | 1 | 0 | 1 | 

| 16 | 8 | 4 | 2 | 1 |

Lalu sampai berapa banyak digit angka? Kita ketahui bahwa penulisan bit pada komputer itu tidak memiliki spasi, titik atau koma semisal 10110100100011 apakah itu semua memiliki satu fungsi? Tentu saja tidak, komputer kita sudah memiliki kemampuan untuk membagi bit-bit tersebut kedalam potongan-potongan yang di sebut dengan nilai/values. Jadi kalau misalnya nilai tersebut terdiri dari beberapa bit bisa aja memiliki peran sendiri-sendiri. Peran inilah yang nanti kita sebut sebagai Type Data. 

Di dalam Javascript ada 6 type data yaitu:

  1. number
  2. string
  3. boolean
  4. object
  5. function 
  6. undefined

Dengan Javascript kita tidak perlu untuk menuliskan bit lagi contoh kita mau membuat tipe data angka kita hanya menuliskan:

Syntax JavaScript

10 / -12000 / 2.93 

Script tersebut memiliki Tipe Data Number atau kita mau membuat string maka kita cukup menulis 'hello' atau string dua "word" maka javascript akan otomatis menerjemahkan ke bentuk bit yang dapat diproses oleh komputer. 

Untuk mengetahui type data apakah berbentuk number, string dll kita dapat menggunakan:

Syntax JavaScript

typeof()

Contoh:

Source JavaScript 

let x = 10

let y = "20"

typeof(x) //number

typeof(y) //string
