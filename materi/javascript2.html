Struktur Dasar JavaScript

 Program JavaScript terdiri dari beberapa block-block dan berjalan dari atas ke bawah dan dari kiri ke kanan jadi tidak akan tertukar dengan program lain, karena JavaScript akan mengerjakan block perblocka dalam menulis JavaScript terdapat dua metode yaitu:

Metode Inline/Internal JavaScript

Metode ini adalah dimana penulisan script dalam satu file dengan dokumen html dan dibungkus dengan tag <script>....</script> biasanya diletakkan sebelum tag </head> atau </body> .

Source Code 

<!DOCTYPE html>

<html>

<head>

 <title>Belajar Struktur Dasar Javascript</title>

 <script type="text/javascript">

   //JavaScript code

document.getElementById("coba").innerHTML = "Saya belajar JavaScript"

 </script>

</head>

<body>

<p id="coba"></p>

</body>

</html>

Output

Saya belajar JavaScript

Penjelasan:

Fungsi tag <script> adalah untuk memberi tau kepada browser bahwa yang kita tulis adalah JavaScript.

Type digunakan untuk menandakan bahwa ini adalah type JavaScript namun dengan ES6 kita penulisan type tersebut bersifat opsional kita bisa menuliskan seperti ini saja:

Syntax JavaScript

<script>

   JavaScript code

</script>

Metode Inline JavaScript jarang sekali digunakan dalam pembuatan website karena jadi satu dengan dokumen html kita akan rumit untuk menambahkan atau merubah script tersebut.

Metode Linked/Eksternal JavaScript

Metode ini digunakan untuk menulis JavaScript di file yang berbeda dengan dokumen html, metode ini tidak membutuhkan tag pembungkus <script>....</script> file yang berisi JavaScript harus berekstensi .js sebagai contoh berikut:

script.js

document.getElementById("coba").innerHTML = "Saya belajar JavaScript"

Untuk dapat digunakan pada dokumen html kita harus memanggil file tersebut dengan cara:

Syntax HTML

<script src="...."</script>

Isi bagian src dengan alamat file JavaScript itu disimpan jika menyimpan file satu folder dengan dokumen html.

Syntax HTML

<script src="script.js"</script>

Jika file tersebut sudah di upload ke hosting dan berbeda folder maka harus menuliskan alamat domain tempat script tersebut disimpan.

Syntax HTML

<script src="https://dupandesdata.github.io/database/script.js"</script>

Syntax tersebut bisa diletakkan sebelum tag </head> atau </body> .

Source Code 

<!DOCTYPE html>

<html>

<head>

 <title>Belajar Struktur Dasar Javascript</title>

<script src="script.js"</script>

</head>

<body>

<p id="coba"></p>

</body>

</html>

Output

Saya belajar JavaScript

Hasilnya sama seperti motode Inline tetapi jika kita ingin mengedit script lebih mudah daripada metode Inline, bayangkan jika menggunakan Inline dan kita tulis didalam index.html gak mungkin didalam file index berisi beberapa baris saja bahkan sampai ribuan baris kita akan rumit untuk mencari script yang akan di edit itulah kenapa metode Inline jarang digunakan.

Memberi Komentar

Komentar berfungsi untuk menandai fungsi dari block script agar mempermudah dalam membaca script dan JavaScript mengabaikan itu berikut dua cara menulis komentar:

  1. // 
  • Teks apa saja yang ditulis setelah // sampai akhir baris dianggap sebagai komentar.
  1. /*.......*/
  • Teks apa saja yang ditulis setelah karakter /* dan */ dianggap sebagai komentar dengan banyak baris (multiline).

Source Code 

<script type="text/javascript">

      // Ini komentar satu baris

      /*

       Ini baris pertama 

       Ini baris kedua

      */

</script>

Case Sensitivity

Seperti halnya dengan program lain JavaScript juga memiliki case-sensitive. Ini berfungsi bahwa jika kita membuat nama variabel, nama funcition, dan lainnya harus bersifat unik tidak boleh sama.

Seperti halnya dengan selector pada CSS karakter sangat sensitif terhadap isi dari nama variabel, contoh kita membuat nama variabel house, House dan HOUSE itu memiliki arti yang berbeda pada JavaScript.

Titik Koma ;

Penulisan titik koma pada akhir perintah JavaScript bersifat opsional namun disarankan untuk menggunakan ; pada akhir perintah.

Tanda Petik 

Ada dua jenis tanda petik yaitu tanda petik satu ' dan tanda petik dua " fungsi dari keduanya adalah untuk membuat kalimat pada JavaScript, namun ketika kita membuat kalimat dengan menggunakan tanda petik satu harus diakhiri dengan tanda petik satu.
