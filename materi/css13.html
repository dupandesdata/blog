CSS Style Link

Dupandes Milenium KD.

30 November 2022

 CSS ini digunakan untuk memberi styling pada elemen link agar terlihat berbeda dengan teks lainnya supaya user mengetahui element link tersebut.

Seperti halnya element pada umumnya kita dapat memberikan styling seperti background, color, font-family dll namun, element link memiliki sytling default tergantung pada statusnya sebagai berikut:

Syntax Keterangan
a:link  Kondisi ketika link normal (belum dikunjungi)
a:visited Kondisi ketika link sudah dikunjungi oleh user
a:hover Kondisi link ketika dihover
a:active Kondisi saat link di klik oleh user

Teks yang mengandung link secara default bergaris bawah dan memiliki warna yang berbeda tergantung pada status kondisinya.

a:link

~ memiliki warna biru dan bergaris bawah.

a:visited

~ memiliki warna ungu dan bergaris bawah.

a:active

~ memiliki warna merah dan bergaris bawah.

Namun sytling tersebut akan hilang ketika kita memberikan warna:

Source Code

a { color: #000; }

Source code diatas adalah membuat semua tag <a> berwarna hitam termasuk a:link, a:visited dan a:active untuk menghilangkan garis bawah pada element link kita bisa menggunakan Text Decoration:

Source Code

a { 

     color: #000;

     text-decoration: none;

}

Kita juga dapat membuat styling sesuka hati seperti berikut: 

Source Code 

a { 

     text-decoration: none;

}

a:link {

  color: green;

}

a:visited {

  color: blue;

}

a:hover {

  color: red;

  text-decoration: underline;

}

a:active {

  color: yellow;

  text-decoration: solid;

}
