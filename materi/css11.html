Text Decoration

Dupandes Milenium KD.

28 November 2022

 Properti CSS ini digunakan untuk membuat/mengatur tampilan dekorasi teks agar terlihat menarik pada halaman website.

Decoration Line

text-decoration-line: value;

Properti ini digunakan untuk mendekorasi garis pada teks, berikut beberapa value yang bisa dilakukan:

Value Output
overline Saya senang belajar CSS
line-through Saya senang belajar CSS
underline Saya senang belajar CSS
overline underline Saya senang belajar CSS

Decoration Color

text-decoration-color: value;

Properti ini digunakan untuk mendekorasi garis warna pada teks, berikut beberapa value yang bisa dilakukan:

Jenis Value
Color red, yellow, green
Hexadecimal #fff, #000, #00ff00
Red-green-blue rgba(108.58.63), rgba(0%.100%.50%)
Hue-saturation-lightness hsl(245.75%.50%)
Hue-saturation-lightness-alpha hsla(245.75%.50%.0.1)

Decoration Style

text-decoration-style: value;

Properti ini digunakan untuk mendekorasi gaya garis pada teks, berikut beberapa value yang bisa dilakukan:

Value Output
solid  Saya senang belajar CSS
double Saya senang belajar CSS
dotted Saya senang belajar CSS
dashed Saya senang belajar CSS
wavy Saya senang belajar CSS
❛❛ Catatan:

Untuk pengguna properti text-decoration-style harus bersamaan dengan text-decoration-line

Decoration Thickness

text-decoration-thickness: value;

Properti ini digunakan untuk mendekorasi ketebalan gaya garis pada teks, berikut beberapa value yang bisa dilakukan:

Value Keterangan
px Pixels
pt Points
vh Viewport

Secara default semua link tag <a> memiliki garisbawa, namun garis bawah tersebut terkesan buruk bagi tampilan seperti dalam tampilan navbar akan terkesan buruk jika memiliki garis bawah pada link untuk menghapus garis bawah tersebut kita bisa menggunakan:

text-decoration: none;

Source Code

a {

  text-decoration: none;

}
