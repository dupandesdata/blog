Box Model

Dupandes Milenium KD.

17 November 2022

Box Model adalah sebuah kotak pembungkus dari setiap elemen HTML, element tersebut di representasikan sebagai box-box dan memahami box ini adalah kunci utama dari sebuah styling tata letak dasar dengan CSS.

Property dan Value CSS

Property CSS adalah sebuah perintah yang diberikan untuk memberi warna, mengubah ukuran dan mengubah bentuk elemen html.

Property dan value diletakkan pada kurang kurawal { } untuk memberi sebuah value jangan lupa untuk menambahkan (dua titik) : pada akhir property dan untuk value diakhiri dengan (titik koma); 

Height

Selector { height: value }

Properti ini digunakan untuk mengatur tinggi element, untuk menggunakan properti hieght harus berbarengan dengan width.

Width

Selector { width: value }

Properti ini digunakan untuk mengatur lebar element, untuk menggunakan properti hieght harus berbarengan dengan height.

Source Code

<style>

div {

  height: 200px;

  width: 200px;

  color: green;

}

</style>

<div></div>

Margin

Selector { margin: value; }

Property margin adalah sebuah perintah untuk memberikan jarak/ruang bagian luar antara satu elemen dengan elemen yang lainnya.

Source Code

<style>

div {

  height: 200px;

  width: 200px;

  margin: 20px;

  color: green;

}

</style>

<div></div>

Padding

Selector { padding: value; }

Properti padding adalah sebuah perintah untuk memberikan jarak/ruang bagian dalam element.

Source Code

<style>

div {

  margin: 20px;

  padding: 20px;

  height: 200px;

  width: 200px;

  color: green;

}

.box {

width: 100px;

height: 100px;

background: coral;

}

</style>

<div>

<div class="box">ini content</div>

</div>

Konsep dasar penulisan value margin atau padding sama sebagai berikut:

  • margin/padding: 10px 5px 5px 10px;
  • margin/padding: 10px 5px 10px;
  • margin/padding: 8px 12px ;

Selain itu margin dan padding juga ditulis seperti berikut:

Padding

  • padding-top: 10px;
  • padding-left: 5px;
  • padding-bottom: 7px;
  • padding-right: 9px;

Margin

  • margin-top: 10px;
  • margin-left: 5px;
  • margin-bottom: 7px;
  • margin-right: 9px;

Penjelasan:

Angka silahkan disesuaikan dengan kebutuhan.

  • padding/margin-top (hanya jarak atas)
  • padding/margin-left (hanya jarak kiri)
  • padding/margin-bottom (hanya jarak bawah)
  • padding/margin-right (hanya jarak kanan)

Contoh saya akan membuat margin top saja sebesar 20px maka kita tidak perlu menulis seperti ini:

Selector { margin: 20px 0 0 0; }

Penjelasan:

Kode diatas berfungsi mengatur jarak luar element bagian atas sebesar 20px dan bawah, kanan dan kiri 0 penulisan seperti itu kurang baik, karena penulisan seperti itu digunakan jika kita ingin memberikan margin atas, bawah, kanan dan kiri namun ketika kita ingin memberikan jarak tertentu saja gunakan:

Padding

  • padding-top: ;
  • padding-left: ;
  • padding-bottom: ;
  • padding-right: ;

Margin 

  • margin-top: ;
  • margin-left: ;
  • margin-bottom: ;
  • margin-right: ;

Border

Properti ini digunakan untuk memberi garis tepi pada element tertentu, untuk membuat border cek disini.

Untuk mengatur ukuran element HTML kita dapat menggunakan satuan sebagai berikut:

Value Keterangan
px Pixels digunakan untuk memberi ukuran fix pada element/font.
pt Points digunakan untuk memberi ukuran font.
vh Viewport digunakan untuk memberi ukuran element/font yang ditampilkan sesuai layar.
rem Rem ukuran font relatif pada elemen root.
% persen digunakan untuk memberikan ukuran element/font sesuai dengan lebar/tinggi element parent.
