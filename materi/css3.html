Property dan Value CSS

Property CSS adalah sebuah perintah yang diberikan untuk memberi warna, mengubah ukuran dan mengubah bentuk elemen html.

Property dan value diletakkan pada kurang kurawal { } untuk memberi sebuah value jangan lupa untuk menambahkan (dua titik) : pada akhir property dan untuk value diakhiri dengan ; 

Syntax CSS

selector {

  property: value;

}

Property dibawah ini adalah jenis property yang wajib dihafal: 

Property:

  1. height = untuk memberi style tinggi.
  2. width = untuk memberi style lebar.
  3. pandding = mengosongkan ruang di sekitar elemen.
  4. margin = membuat jarak pada elemen.
  5. float = memaksa elemen ke kiri atau kanan. 

Value adalah tempat dimana kita menulis warna, tinggi dan lebar, besar dan kecil suatu konten.

Value:

  1. px = pixels digunakan untuk memberi ukuran fix element.
  2. pt = points digunakan untuk memberi ukuran font.
  3. vh = viewport digunakan untuk memberi ukuran element/text yang ditampilkan sesuai layar.
  4. em = rem digunakan untuk memberi ukuran element/text yang menyesuaikan dengan layar.
  5. % = persen digunakan untuk memberikan ukuran element/text sesuai layar. 

Property dan value 

adalah sebuah syntax CSS yang bisa digunakan untuk property atau bisa juga digunakan sebagai value sebagai berikut:

  1. top = atas
  2. bottom = bawah
  3. right = kanan
  4. left = kiri

Bertindak sebagai property:

Syntax Property

top: value;

bottom: value;

right: value;

left: value;

Syntax CSS 

selector {

  top: 10px;

}

Penjelasan:

Syntax CSS diatas berfungsi memberikan jarak bagian atas dari element tertentu.

Bertindak sebagai value

Syntax Value

nama property: top;

nama property: bottom;

nama property: right;

nama property: left;

Syntax CSS

selector {

  float: right;

}

Penjelasan:

Syntax CSS diatas berfungsi untuk menampilkan element di kanan halaman.
