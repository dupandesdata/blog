Tabel Border

Ini digunakan untuk menentukan batas sel kita bisa menggunakan CSS selector grouping:

Source Code 

table,
th,
td {
  border: 1px solid;
}

Tentang properti border 

Border Ganda

Hal ini di karenakan element <tabel>, <th> dan <td> adalah element yang berbeda, untuk menghapus border ganda kita bisa menggunakan properti berikut:

border-collapse: collapse;

Source Code

table,
th,
td {
  border: 1px solid;
  border-collapse: collapse;
}

Tabel Layout

table-layout: auto | fixed

Properti ini digunakan untuk tata letak tabel sel, baris dan kolom.

Penjelasan:

auto

Value ini adalah value default, ketika tabel tidak diberi layout maka secara otomatis browser menggunakan auto (algoritme) untuk tata letak tabel. Lebar tabel akan disesuaikan dengan konten didalam sel.

fixed

Value ini digunakan untuk mengatur lebar sel yang diatur dengan lebar sel dibaris pertama, sedangkan sel dibaris berikutnya tidak memengaruhi lebar kolom.

Table Alignment

text-align: left | center | right

Properti yang digunakan untuk mengatur konten horizontal, dalam penempatan konten <th> atau <td> ditempatkan di kiri, di kanan atau di tengah. 


Vertical Alignment

vertical-align: top | center | bottom

Properti yang digunakan untuk mengatur konten vertikal, dalam penempatan konten <th> atau <td> ditempatkan di atas, di tengah atau di bawah.
