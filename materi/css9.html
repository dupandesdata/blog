Height dan Width

Dupandes Milenium KD.

23 November 2022

Height

height: value;

Properti ini tidak mempengaruhi padding atau margin, value yang bisa dilakukan hanya angka positif sedangkan angka negatif seperti height: -10px tidak diterima. Properti height tidak mempengaruhi element lain kecuali dengan selector yang sama. 

Width

width: value;

Properti ini digunakan untuk menentukan lebar element, properti ini merupakan bagian dari area "konten" dan mempengaruhi ( model kotak ). Value yang bisa dilakukan hanya angka positif sedangkan angka negatif seperti width: -10px tidak diterima. Properti width tidak mempengaruhi element lain kecuali dengan selector yang sama.

Value

Properti height dan width memiliki value sebagai berikut:

Jenis Value Keterangan
auto menghitung ketinggian sesuai dengan konten yang ada didalam elemen yang ditentukan
length Menentukan tinggi atau lebar dalam satuan px, cm, vh, pt dll
% Menentukan tinggi atau lebar dalam bentuk persentase yang diukur dari elemeet parent

Min & Max Konten

Properti ini digunakan untuk tinggi/lebar minimum dan maksimum element tertentu, properti ini juga berkaitan dengan responsive jika kita memiliki elemet dengan lebar (700px) maka secara otomatis browser akan menambahkan scrollbar jika dibuka dengan menggunakan ukuran layar di bawah 700px.

Dengan menggunakan max-width, element tersebut jika dibuka dengan ukuran layar kecil atau kurang dari 700px maka yang terjadi lebar element yang tadinya 700px menjadi 100% dan ketika dibuka dengan ukuran layar melebihi 700px maka akan memiliki lebar 700px.

min-height: value;

max-height: value;

max-width: value;

min-width: value;
