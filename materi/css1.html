 Cascading Style Shee adalah bahasa design (Styling Language) yang berisi sekumpulan format untuk mengatur bagaimana styling tag-tag HTML pada dokumen. Berikut ini rekomendasi CSS cara menampilkan styling pada tag HTML:

Embedded CSS

Styling ini diletakkan dalam satu block element HTML. 

Syntax CSS

style="  "

index.html

<p style="color: blue; font-size: 10pt"> ini paragraf</p>

penjelasan:

CSS jenis Embedded adalah penulisan styling langsung pada block dan hanya berlaku pada block tersebut saja.

Inline/Internal CSS

Styling ini ditulis secara langsung dalam dokumen HTML namun harus dibungkus oleh tag <style>....</style>.

Syntax CSS

<style>

selector {   }

 </style>

Linked/ Eksternal CSS

Styling ini diletakkan di file yang berbeda yang berekstensi .css penulisan styling CSS Linked sama seperti Inline namun tidak di bungkus dengan tag <style>....</style> untuk dapat menggunakan CSS ini kita harus memanggil file tersebut dengan tag <link/>.

style.css

.h1 {

  text-align: center;

  color: #000;

}

Tag <link/> harus diletakkan sebelum tag </head>

Syntax HTML 

<link href="...." rel="stylesheet"/>

index.html

<!DOCTYPE html>

<html>

  <head>

    <title>Dupan Code | </title>

    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

    <link href="style .css" rel="stylesheet"/>

  </head>

  <body>

    <h1>Dupan Code</h1<

  </body>

</html>

Penjelasan:

Jika kita menjalankan dokumen html di lokalhost dan file CSS tersebut satu folder dengan dokumen html maka kita bisa menuliskan nama filenya saja dalam href, tetapi jika file tersebut sudah di upload ke server hosting contohnya github maka kita harus menuliskan alamat file tersebut.

Eksternal CSS 

<link href="https://dupandesdata.github.io/database/web/css/allstyle.css" rel="stylesheet"/>

Dalam studi kasus ini style langsung mengenai tag html, ini biasanya digunakan untuk memberi style yang sama disemua tag tersebut. Cara ini mempermudah proses pembuatan website kita tidak perlu memanggil style tersebut setiap kali menuliskan elemen tag h1.

CSS memiliki tiga type selector

  1. ID selector 
  2. Class selector 
  3. Grouping selector 

Apa perbedaan antara class selector, id selector dan grouping selector?

ID selector kita harus menggunakan atribut id="  " pada elemen html, id selector ini memiliki sifat unik di satu halaman biasanya digunakan untuk memberi style unik pada tag yang di berikan atribut tersebut.

Syntax HTML

<p id="myName">Dupandes</p>

Class selector adalah sebuah class yang sudah kita berikan style yang diinginkan pada lebih dari satu tag, untuk memanggil class selector ini kita menggunakan atribut class="  " pada elemen html.

Syntax HTML

<div class="prof">Web Development</div>

Penjelasan:

Walaupun digunakan pada tad yang berada namun mempunyai class selector yang sama ini akan menghasilkan output styling yang sama.

Grouping Selector adalah sebuah teknik menulis CSS untuk memberi tag html dengan style yang sama, untuk cara penulisan CSS group yaitu dengan menggunakan tanda , pada akhir selector.

Penulisan Grouping dengan menggunakan tag:

h1, h2, h3, p, div {

color: red;

}

Penulisan HTML:

<h1>Saya</h1>

<h2>Sedang belajar</h2>

<h3>CSS</h3>

<p>Ini menyenangkan</p>

<div>Sekali</div>

Penjelasan:

Walaupun dokumen html ditulis dengan tag yang berbeda tetapi dengan menggunakan CSS Grouping memiliki styling yang sama yaitu warna font red.
