 Cascading Style Shee adalah bahasa design (Styling Language) yang berisi sekumpulan format untuk mengatur bagaimana styling tag-tag HTML pada dokumen. Berikut ini rekomendasi CSS cara menampilkan styling pada tag HTML:

Embedded CSS

style="  "

Styling ini diletakkan dalam satu block element HTML. 

Source Code

<p style="color: blue; font-size: 10pt"> ini paragraf 1</p>

penjelasan:

CSS jenis Embedded adalah penulisan styling langsung pada block dan hanya berlaku pada block tersebut saja.

Inline/Internal CSS

<style>....</style>

Styling ini ditulis secara langsung dalam dokumen HTML namun harus dibungkus oleh tag <style>.

Source Code

<style>

selector {

color: blue;

}

</style>

Linked/Eksternal CSS

<link href="...." rel="stylesheet"/>

Styling ini diletakkan di file yang berbeda yang berekstensi .css penulisan styling CSS Linked sama seperti Inline namun tidak di bungkus dengan tag <style>....</style> untuk dapat menggunakan CSS ini kita harus memanggil file tersebut dengan tag <link/>.

style.css

.h1 {

  text-align: center;

  color: #000;

}

Tag <link/> harus diletakkan sebelum tag </head>

index.html

  <head>

    <link href="style.css" rel="stylesheet"/>

  </head>

Penjelasan:

Jika kita menjalankan dokumen html di lokalhost dan file CSS tersebut satu folder dengan dokumen html maka kita bisa menuliskan nama filenya saja dalam href, tetapi jika file tersebut sudah di upload ke server hosting contohnya github maka kita harus menuliskan alamat file tersebut.

Source Code 

<link href="https://dupandesdata.github.io/database/web/css/allstyle.css" rel="stylesheet"/>
