Text Transformation

Dupandes Milenium KD.

29 November 2022

 Properti ini digunakan untuk menampilkan teks element menjadi huruf besar atau huruf kecil semua atau mengawali kata/paragraf dengan menggunakan huruf kapital (capitalize).

Capitalize

text-transform: capitalize;

Properti ini digunakan mengawali setiap kata dengan huruf kapital pada huruf pertama dalam elemet tertentu, tanda baca dan simbol akan diabaikan.

Lowercase

text-transform: lowercase

Properti ini akan merubah semua huruf dalam element tertentu menjadi huruf kecil semua.

Uppercase

text-transform: uppercase

Properti ini akan merubah semua huruf dalam element tertentu menjadi huruf besar semua.

None

text-transform: none

Properti ini digunakan untuk menampilkan teks persis seperti yang ditulis.

Inherit

text-transform: inherit

Properti ini digunakan untuk mengatur kapitalisasi teks induk.

❛❛ Catatan:

capitalize kurang didukung oleh CSS 1 dan CSS 2.1 yang mengakibatkan perbedaan pada setiap browser dalam menghitung huruf pertama, baik menggunakan Webkit atau Gecko tidak bisa membedakan simbol huruf seperti ⓐ dianggap sebagai huruf asli. Terdapat beberapa kasus aneh seperti pada browser Internet Explorer 9, browser tersebut yang paling dekat dengan definisi CSS 2.

Mengkapitalisasi huruf pertama setelah tanda kutip tunggal atau ganda dan setelah tanda hubung, pada huruf pertama yang diawali dengan angka tidak akan dikapitalisasi tanggal seperti "4 januari 2022" tidak akan berubah menjadi "4 Januari 2022".

CSS tidak mendukung kapitalisasi pada judul film, buku, lagu dll, yang di mana penulisan  seperti "Raiders of the Lost Ark").  Namun, ada solusi JavaScript untuk kapitalisasi judul, yaitu menggunakan toTitleCase() milik David Gouch .
