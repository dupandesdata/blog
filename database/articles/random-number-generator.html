Penjelasan Aplikasi Random Number Generator

Random Number Generator adalah aplikasi web sederhana yang memungkinkan pengguna untuk menghasilkan angka acak dalam rentang yang ditentukan, dengan dua mode operasi: Random (mengizinkan duplikat) dan Unik (tanpa duplikat). Aplikasi ini dibangun menggunakan Vue.js 3 untuk antarmuka yang responsif, Tailwind CSS untuk desain modern dengan gradien dan animasi, serta animasi kustom untuk transisi yang mulus. Aplikasi ini dirancang untuk memberikan pengalaman pengguna yang intuitif dengan pratinjau hasil yang menarik dan informasi tambahan seperti jumlah angka unik yang tersisa dalam mode Unik.

Fitur utama aplikasi meliputi:

  • Generasi angka acak berdasarkan rentang minimum dan maksimum yang ditentukan pengguna.
  • Dua mode: Random (boleh duplikat) dan Unik (setiap angka hanya muncul sekali).
  • Antarmuka responsif dengan desain estetis, termasuk gradien warna dan efek hover.
  • Animasi fade-in untuk menampilkan hasil angka acak.
  • Pelacakan angka unik yang tersisa dalam mode Unik untuk mencegah kehabisan angka.
  • Validasi otomatis untuk menonaktifkan tombol saat tidak ada lagi angka unik yang tersedia.

Aplikasi ini cocok untuk berbagai keperluan, seperti undian, simulasi, permainan, atau kebutuhan lain yang memerlukan angka acak atau unik.

Cara Menggunakan Random Number Generator

Berikut adalah panduan langkah demi langkah untuk menggunakan aplikasi ini secara detail:

1. Akses Aplikasi

  • Buka aplikasi melalui browser modern (Chrome, Firefox, Safari) yang mendukung JavaScript dan Tailwind CSS.
  • Aplikasi akan menampilkan antarmuka dengan judul "Random Generator", kolom input untuk minimum dan maksimum, dropdown untuk memilih mode, tombol untuk menghasilkan angka, dan area untuk menampilkan hasil.

2. Atur Rentang Angka

  • Minimum: Masukkan angka minimum untuk rentang di kolom bertanda "Minimum" (default: 1).
    • Contoh: 1 untuk memulai dari angka 1.
  • Maximum: Masukkan angka maksimum untuk rentang di kolom bertanda "Maximum" (default: 100).
    • Contoh: 100 untuk membatasi hingga angka 100.
  • Kolom ini hanya menerima angka (integer). Jika Anda mengubah nilai minimum atau maksimum, daftar angka unik yang sudah digunakan akan direset secara otomatis untuk memulai dari awal.

3. Pilih Mode

  • Di dropdown bertanda "Mode", pilih salah satu opsi:
    • Random (boleh duplikat): Menghasilkan angka acak dalam rentang tanpa mempedulikan apakah angka tersebut sudah muncul sebelumnya.
    • Unik (hanya sekali): Menghasilkan angka acak yang belum pernah muncul sebelumnya dalam sesi tersebut. Aplikasi akan melacak angka yang sudah digunakan.
  • Jika Anda mengubah mode, daftar angka unik yang sudah digunakan akan direset.

4. Hasilkan Angka

  • Klik tombol "Generate Number" untuk menghasilkan angka acak sesuai pengaturan.
  • Hasil akan ditampilkan di area di bawah tombol dengan animasi fade-in, menunjukkan:
    • Your random number: Angka acak yang dihasilkan (misalnya, 42).
    • Jika mode Unik, informasi tambahan "Tersisa: X angka unik" akan ditampilkan, menunjukkan berapa banyak angka yang masih tersedia dalam rentang.
  • Jika mode Unik dan semua angka dalam rentang sudah digunakan:
    • Tombol akan dinonaktifkan (opacity 50% dan cursor tidak aktif).
    • Klik tombol akan memunculkan peringatan "Semua angka unik sudah digunakan!".

5. Ulangi atau Sesuaikan

  • Untuk menghasilkan angka baru, klik kembali tombol "Generate Number".
  • Anda dapat:
    • Mengubah nilai Minimum atau Maksimum untuk rentang baru (ini akan mereset daftar angka unik).
    • Beralih antara mode Random dan Unik (juga mereset daftar angka unik).
  • Hasil sebelumnya akan digantikan dengan angka baru setiap kali tombol ditekan.

6. Gunakan Hasil

  • Angka yang dihasilkan dapat dicatat secara manual atau digunakan untuk keperluan seperti undian, pengujian, atau permainan.
  • Aplikasi tidak menyediakan fitur untuk menyalin atau menyimpan hasil, jadi salin angka secara manual jika diperlukan.

Fitur Utama

  • Generasi Angka Acak: Menghasilkan angka dalam rentang yang ditentukan dengan dua mode (Random dan Unik).
  • Mode Unik: Melacak angka yang sudah digunakan untuk memastikan tidak ada duplikat dalam sesi.
  • Desain Estetis: Menggunakan gradien warna, efek hover, dan animasi fade-in untuk pengalaman visual yang menarik.
  • Validasi Otomatis: Menonaktifkan tombol saat tidak ada angka unik yang tersisa dalam mode Unik.
  • Antarmuka Responsif: Didesain dengan Tailwind CSS untuk mendukung penggunaan di desktop dan perangkat seluler.
  • Reset Otomatis: Mengosongkan daftar angka unik saat mode, minimum, atau maksimum diubah.

Catatan Penting

  • Validasi Input:
    • Kolom Minimum dan Maksimum harus berisi angka (integer). Nilai non-angka akan diabaikan oleh Vue.js.
    • Pastikan Minimum lebih kecil atau sama dengan Maksimum untuk hasil yang logis (aplikasi tidak memvalidasi ini secara eksplisit).
  • Mode Unik:
    • Jumlah angka yang tersedia adalah max - min + 1. Jika semua angka digunakan, tombol akan dinonaktifkan hingga rentang atau mode diubah.
    • Daftar angka unik disimpan dalam sesi browser dan akan direset saat halaman dimuat ulang.
  • Performa:
    • Generasi angka dilakukan di sisi klien menggunakan Math.random(), sehingga tidak memerlukan koneksi internet setelah aplikasi dimuat.
    • Animasi dan transisi ringan, cocok untuk perangkat dengan spesifikasi rendah.
  • Browser:
    • Gunakan browser modern untuk memastikan kompatibilitas dengan Vue.js 3 dan Tailwind CSS.
    • Beberapa browser lama mungkin tidak mendukung fitur seperti gradien atau animasi CSS.
  • Keterbatasan:
    • Tidak ada fitur untuk menyimpan atau mengekspor hasil angka.
    • Tidak ada validasi untuk memastikan rentang logis (misalnya, min > max).
