Penjelasan Aplikasi Al-Qur'an Digital

Al-Qur'an Digital adalah aplikasi web berbasis Vue.js yang dirancang untuk memudahkan pengguna dalam membaca, mencari, dan mempelajari Al-Qur'an secara digital. Aplikasi ini menyediakan daftar surah, detail ayat beserta teks Arab, transliterasi, terjemahan dalam bahasa Indonesia, serta fitur audio untuk mendengarkan bacaan ayat. Aplikasi ini juga dilengkapi dengan fitur pencarian, navigasi antar surah, dan penyimpanan bacaan terakhir menggunakan localStorage. Desainnya responsif, menggunakan Tailwind CSS untuk tampilan modern dan nyaman di berbagai perangkat.

Aplikasi ini mengambil data dari API eQuran (https://equran.id/api/v2), yang menyediakan informasi lengkap tentang surah dan ayat Al-Qur'an. Fitur-fitur utama meliputi:

  • Daftar Surah: Menampilkan semua surah dengan nama Latin, nama Arab, arti, tempat turun, dan jumlah ayat.
  • Pencarian Surah: Memungkinkan pengguna mencari surah berdasarkan nama Arab atau Latin.
  • Detail Surah: Menampilkan teks Arab, transliterasi, terjemahan, dan audio untuk setiap ayat.
  • Navigasi Surah: Tombol untuk berpindah ke surah sebelumnya atau berikutnya.
  • Bacaan Terakhir: Menyimpan surah dan ayat terakhir yang dibaca, ditampilkan dalam popup saat membuka aplikasi.
  • Audio Ayat: Memungkinkan pengguna memutar audio bacaan ayat.
  • Tampilan Responsif: Disesuaikan untuk desktop dan perangkat mobile.
  • Transisi Halus: Menggunakan Vue Router dengan animasi fade untuk perpindahan halaman.

Cara Menggunakan Aplikasi Secara Detail

Berikut adalah panduan langkah demi langkah untuk menggunakan aplikasi Al-Qur'an Digital:

1. Mengakses Aplikasi

  • Buka aplikasi di browser (pastikan terhubung ke internet untuk mengambil data dari API).
  • Halaman utama akan menampilkan daftar surah dan header bertuliskan "Al-Qur'an Digital".
  • Jika ada bacaan terakhir yang tersimpan, popup akan muncul menampilkan nama surah, nomor ayat, dan tombol "Lanjutkan Membaca".

2. Mencari Surah

  • Di halaman utama, terdapat kolom pencarian dengan placeholder "Cari Surah...".
  • Ketik nama surah (misalnya, "Al-Fatihah" atau "الفاتحة") untuk memfilter daftar surah.
  • Daftar surah akan diperbarui secara otomatis berdasarkan kata kunci yang dimasukkan.

3. Melihat Daftar Surah

  • Setiap surah ditampilkan dalam bentuk kartu (card) dengan informasi:
    • Nama Latin (misalnya, Al-Fatihah).
    • Nama Arab (misalnya, الفاتحة).
    • Arti (misalnya, Pembukaan).
    • Tempat Turun (Mekah/Madinah) dan Jumlah Ayat.
  • Klik kartu surah untuk masuk ke halaman detail surah.

4. Membaca Detail Surah

  • Halaman detail surah menampilkan:
    • Judul Surah: Nama Latin dan Arab di bagian atas.
    • Deskripsi: Informasi tentang surah (misalnya, sejarah atau konteks).
    • Daftar Ayat: Setiap ayat menampilkan:
      • Teks Arab: Ditampilkan dengan font besar dan arah kanan-ke-kiri (RTL).
      • Nomor Ayat: Dalam ikon khusus.
      • Transliterasi dan Terjemahan: Muncul saat tombol keterangan diklik.
      • Tombol Audio: Ikon segitiga untuk memutar/pause audio ayat.
      • Tombol Keterangan: Ikon untuk menampilkan/sembunyikan transliterasi dan terjemahan.
  • Navigasi:
    • Tombol Surat Sebelumnya (jika bukan surah pertama).
    • Tombol Surat Selanjutnya (jika bukan surah terakhir).
  • Saat menggulir, aplikasi akan melacak ayat yang sedang dibaca dan menyimpannya sebagai bacaan terakhir.

5. Menggunakan Fitur Bacaan Terakhir

  • Aplikasi secara otomatis menyimpan surah dan ayat terakhir yang dibaca menggunakan localStorage.
  • Saat membuka aplikasi, popup akan muncul jika ada bacaan terakhir, menampilkan:
    • Nama surah.
    • Nomor ayat.
    • Tombol Lanjutkan Membaca untuk langsung menuju ayat tersebut.
  • Untuk menutup popup, klik tombol × di pojok kanan atas.
  • Bacaan terakhir akan dihapus saat pindah ke surah lain menggunakan tombol navigasi.

6. Memutar Audio Ayat

  • Klik ikon segitiga (play) di samping ayat untuk memutar audio bacaan.
  • Klik lagi untuk pause audio.
  • Audio hanya memutar satu ayat pada satu waktu.

7. Menampilkan Keterangan Ayat

  • Klik ikon keterangan (kotak dengan garis) untuk menampilkan transliterasi (teks Latin) dan terjemahan bahasa Indonesia.
  • Klik lagi untuk menyembunyikan keterangan.
  • Keterangan ditampilkan hanya untuk ayat yang dipilih.

8. Navigasi Antar Surah

  • Gunakan tombol Surat Sebelumnya atau Surat Selanjutnya di bagian bawah halaman detail surah.
  • Klik tombol ini akan menghapus bacaan terakhir dan menggulir ke atas halaman surah baru.

9. Responsivitas

  • Pada perangkat mobile (lebar < 900px), kartu surah ditampilkan dalam satu kolom.
  • Pada desktop (lebar ≥ 900px), kartu surah ditampilkan dalam dua kolom.
  • Tata letak ayat dan elemen lain tetap rapi di semua ukuran layar.

10. Keluar atau Menutup Aplikasi

  • Tutup tab browser untuk keluar.
  • Bacaan terakhir akan tetap tersimpan dan muncul saat membuka aplikasi kembali.

Catatan Teknis

  • API: Aplikasi menggunakan API eQuran untuk mengambil data surah dan ayat.
  • Vue.js: Framework utama untuk logika aplikasi, dengan Vue Router untuk navigasi.
  • Tailwind CSS: Untuk styling responsif dan modern.
  • Axios: Untuk mengambil data dari API.
  • localStorage: Untuk menyimpan bacaan terakhir.
  • IntersectionObserver: Untuk melacak ayat yang sedang dibaca.
  • Audio: Menggunakan elemen HTML Audio untuk memutar bacaan ayat.
