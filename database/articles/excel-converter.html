Penjelasan Aplikasi Excel Converter

Excel Converter adalah aplikasi web yang dirancang untuk mengonversi file Excel (.xlsx, .xls) ke berbagai format keluaran, termasuk JSON, HTML Table, PDF, dan PNG. Aplikasi ini dibangun menggunakan Vue.js 3 untuk antarmuka yang responsif, Tailwind CSS untuk desain modern, serta pustaka seperti XLSX untuk memproses file Excel, html2pdf.js, dan html2canvas untuk menghasilkan file PDF dan PNG. Aplikasi ini memungkinkan pengguna untuk mengunggah file Excel, melihat pratinjau data dalam bentuk tabel, dan mengekspornya ke format yang diinginkan dengan opsi untuk menambahkan header dan footer pada keluaran PDF dan PNG.

Fitur utama aplikasi meliputi antarmuka drag-and-drop untuk mengunggah file, pratinjau data secara real-time, animasi transisi yang halus, dan kemampuan untuk menyesuaikan ekspor dengan header/footer. Aplikasi ini cocok untuk pengguna yang perlu mengubah data Excel ke format lain untuk keperluan pengembangan, presentasi, atau dokumentasi.

Cara Menggunakan Excel Converter

Berikut adalah panduan langkah demi langkah untuk menggunakan aplikasi ini secara detail:

1. Akses Aplikasi

  • Buka aplikasi melalui browser modern (Chrome, Firefox, Safari) yang mendukung JavaScript dan pustaka yang digunakan.
  • Aplikasi akan menampilkan antarmuka dengan judul "Excel Converter", deskripsi singkat, dan area drag-and-drop untuk mengunggah file Excel.

2. Unggah File Excel

  • Klik area bertanda "Select Excel file" atau seret dan lepaskan file Excel (.xlsx atau .xls) ke area tersebut.
  • Setelah file dipilih, aplikasi akan secara otomatis membaca isi file menggunakan pustaka XLSX.
  • Data dari lembar kerja pertama dalam file Excel akan ditampilkan sebagai tabel pratinjau, dengan baris pertama dianggap sebagai header kolom.

3. Lihat Pratinjau Data

  • Jika file berhasil diunggah, aplikasi akan menampilkan:
    • Tabel pratinjau yang menunjukkan data dari file Excel, dengan header kolom di bagian atas dan baris data di bawahnya.
    • Tombol format (JSON, HTML, PDF, PNG) untuk memilih jenis keluaran.
  • Tabel pratinjau akan muncul dalam format yang rapi dan dapat digulir jika datanya luas.

4. Pilih Format Keluaran

  • Pilih format keluaran dengan mengklik salah satu tombol berikut:
    • JSON: Menghasilkan file JSON yang mewakili data tabel sebagai array objek.
    • HTML: Menghasilkan kode HTML untuk tabel yang sesuai dengan data Excel.
    • PDF: Menghasilkan file PDF dari tabel pratinjau, dengan opsi untuk menambahkan header dan footer.
    • PNG: Menghasilkan gambar PNG dari tabel pratinjau, juga dengan opsi header dan footer.
  • Tombol format yang dipilih akan disorot dengan warna biru, sementara yang lain berwarna abu-abu.

5. Konfigurasi Header dan Footer (Opsional, untuk PDF dan PNG)

  • Jika format PDF atau PNG dipilih, opsi "Show Header/Footer" akan muncul.
  • Centang kotak Show Header/Footer untuk mengaktifkan input header dan footer.
  • Masukkan teks untuk:
    • Header: Teks yang akan muncul di atas tabel (misalnya, "Laporan Penjualan 2025").
    • Footer: Teks yang akan muncul di bawah tabel (misalnya, "Dibuat oleh Dupan Code").
  • Teks header dan footer akan ditampilkan dalam pratinjau secara real-time dengan animasi slide saat opsi diaktifkan atau dinonaktifkan.

6. Lihat Kode (untuk JSON dan HTML)

  • Jika format JSON atau HTML dipilih, aplikasi akan menampilkan kode yang dihasilkan dalam elemen <pre> dengan latar belakang gelap.
    • Untuk JSON, kode akan berupa array objek JavaScript, di mana setiap objek mewakili satu baris data dengan header sebagai kunci.
    • Untuk HTML, kode akan berupa struktur tabel HTML dengan elemen <thead> dan <tbody> yang mencerminkan data Excel.
  • Kode ini ditampilkan dengan animasi fade untuk transisi yang halus.

7. Ekspor Data

  • Klik tombol "Export as [Format]" untuk mengunduh file dalam format yang dipilih.
  • Proses ekspor akan berjalan sebagai berikut:
    • JSON: Mengunduh file .json yang berisi data dalam format JSON.
    • HTML: Mengunduh file .html yang berisi kode tabel HTML.
    • PDF: Menggunakan html2pdf.js untuk menghasilkan file PDF dari area pratinjau (termasuk header/footer jika diaktifkan).
    • PNG: Menggunakan html2canvas untuk mengonversi area pratinjau menjadi gambar PNG yang diunduh.
  • File akan diunduh dengan nama default seperti export.json, export.html, export.pdf, atau export.png.

8. Ulangi Proses

  • Untuk mengonversi file lain, unggah file Excel baru. Pratinjau dan pengaturan sebelumnya akan direset secara otomatis.
  • Anda juga dapat mengubah format keluaran atau mengedit header/footer tanpa perlu mengunggah ulang file.

Fitur Utama

  • Konversi Multi-Format: Mendukung ekspor ke JSON, HTML, PDF, dan PNG dari file Excel.
  • Pratinjau Real-Time: Menampilkan data Excel dalam tabel yang rapi sebelum ekspor.
  • Header dan Footer: Opsi untuk menambahkan teks header dan footer pada keluaran PDF dan PNG.
  • Animasi Transisi: Menggunakan animasi fade dan slide untuk pengalaman pengguna yang halus.
  • Antarmuka Responsif: Didesain dengan Tailwind CSS untuk tampilan yang menarik dan kompatibel dengan berbagai perangkat.
  • Drag-and-Drop: Mendukung unggah file dengan seret dan lepas untuk kemudahan penggunaan.
  • Error Handling: Secara implisit menangani file yang tidak valid dengan tidak menampilkan pratinjau jika file gagal diproses.

Catatan Penting

  • Format File: Hanya file Excel dengan ekstensi .xlsx atau .xls yang didukung.
  • Ukuran File: Pastikan file tidak terlalu besar untuk menghindari masalah performa, tergantung pada kemampuan browser.
  • Koneksi Internet: Tidak diperlukan koneksi internet setelah pustaka dimuat, karena pemrosesan dilakukan secara lokal di browser.
  • Browser: Gunakan browser modern untuk memastikan kompatibilitas dengan pustaka seperti XLSX dan html2canvas.
  • Kualitas PDF/PNG: Kualitas keluaran PDF dan PNG bergantung pada pengaturan pustaka (misalnya, skala html2canvas diatur ke 2 untuk ketajaman).
