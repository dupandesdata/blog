Penjelasan Aplikasi OCR with Optional Cropping

OCR with Optional Cropping adalah aplikasi web yang memungkinkan pengguna untuk mengunggah gambar, secara opsional memotong (crop) bagian tertentu dari gambar, dan mengonversi teks dalam gambar tersebut menjadi teks digital menggunakan teknologi Optical Character Recognition (OCR). Aplikasi ini dibangun menggunakan Vue.js 2 untuk logika antarmuka, Tesseract.js untuk pemrosesan OCR, Cropper.js untuk fitur pemotongan gambar, dan Tailwind CSS untuk desain yang modern dan responsif.

Fitur utama aplikasi meliputi:

  • Pengunggahan gambar dari perangkat lokal.
  • Pemotongan gambar secara interaktif untuk fokus pada area tertentu.
  • Konversi teks dari gambar (baik asli maupun yang sudah dipotong) menjadi teks digital.
  • Antarmuka pengguna yang intuitif dengan pratinjau gambar dan hasil teks.
  • Desain responsif yang mendukung penggunaan di berbagai perangkat.

Aplikasi ini ideal untuk pengguna yang perlu mengekstrak teks dari gambar, seperti dokumen, tanda, atau catatan, dengan opsi untuk mempersempit area pemrosesan OCR guna meningkatkan akurasi.

Cara Menggunakan OCR with Optional Cropping

Berikut adalah panduan langkah demi langkah untuk menggunakan aplikasi ini secara detail:

1. Akses Aplikasi

  • Buka aplikasi melalui browser modern (Chrome, Firefox, Safari) yang mendukung JavaScript dan CDN yang digunakan.
  • Aplikasi akan menampilkan antarmuka dengan judul "Image OCR by DUPANDES", area untuk mengunggah gambar, dan tombol untuk memulai konversi teks.

2. Unggah Gambar

  • Klik kolom input bertanda "Upload Image" atau seret dan lepaskan file gambar (format seperti PNG, JPG, dll.) ke area tersebut.
  • Setelah gambar diunggah:
    • Gambar akan ditampilkan di antarmuka sebagai pratinjau.
    • Pesan "Click the image to enable cropping" akan muncul di bawah gambar untuk mengarahkan pengguna ke fitur pemotongan.
  • Catatan: Pastikan gambar memiliki teks yang jelas untuk hasil OCR yang optimal.

3. (Opsional) Potong Gambar

  • Jika Anda ingin memfokuskan OCR pada bagian tertentu dari gambar:
    • Klik gambar untuk mengaktifkan mode pemotongan.
    • Area gambar akan menjadi interaktif, dengan alat Cropper.js yang memungkinkan Anda menyeret dan mengatur kotak pemotongan.
    • Dua tombol akan muncul:
      • Crop Image (hijau): Klik untuk menyimpan area yang dipotong. Gambar yang sudah dipotong akan ditampilkan di bawah dengan judul "Cropped Image:".
      • Cancel (merah): Klik untuk membatalkan pemotongan dan kembali ke gambar asli.
  • Jika tidak ingin memotong, lewati langkah ini dan lanjutkan ke konversi teks.

4. Konversi Gambar ke Teks

  • Klik tombol "Convert to Text" untuk memulai proses OCR.
    • Jika gambar telah dipotong, OCR akan memproses gambar yang dipotong (croppedImageUrl).
    • Jika tidak ada pemotongan, OCR akan memproses gambar asli (imageUrl).
  • Selama pemrosesan:
    • Tombol akan berubah menjadi "Converting..." dan dinonaktifkan untuk mencegah klik berulang.
    • Tesseract.js akan menganalisis gambar untuk mengekstrak teks.
  • Jika berhasil:
    • Teks yang diekstrak akan ditampilkan di area berjudul "Converted Text:" dalam kotak abu-abu.
  • Jika gagal:
    • Pesan "Error occurred during text conversion." akan ditampilkan.
    • Periksa log konsol browser untuk detail kesalahan.

5. Tinjau Hasil

  • Hasil teks dapat disalin secara manual dari area Converted Text untuk digunakan di aplikasi lain.
  • Jika hasil OCR tidak akurat:
    • Pastikan gambar memiliki teks yang jelas dan cukup terang.
    • Coba potong gambar untuk fokus pada area teks saja, lalu ulangi konversi.
  • Untuk mengunggah gambar baru, ulangi proses dari langkah 2. Gambar sebelumnya akan diganti, dan hasil teks akan direset.

6. Ulangi atau Eksperimen

  • Anda dapat:
    • Mengunggah gambar baru untuk diproses.
    • Memotong ulang gambar yang sama dengan mengklik gambar dan menyesuaikan kotak pemotongan.
    • Mengulangi konversi teks setelah pemotongan baru.
  • Aplikasi akan secara otomatis menghapus data pemotongan sebelumnya saat gambar baru diunggah.

Fitur Utama

  • Pengunggahan Gambar: Mendukung berbagai format gambar melalui input file.
  • Pemotongan Interaktif: Menggunakan Cropper.js untuk memungkinkan pemotongan gambar secara presisi.
  • Konversi OCR: Menggunakan Tesseract.js untuk mengekstrak teks dari gambar dengan akurasi tinggi.
  • Pratinjau Visual: Menampilkan gambar asli, gambar yang dipotong, dan hasil teks dalam antarmuka yang rapi.
  • Antarmuka Responsif: Didesain dengan Tailwind CSS untuk tampilan yang menarik di desktop dan perangkat seluler.
  • Efek Hover: Tombol dan gambar memiliki animasi hover untuk pengalaman pengguna yang lebih interaktif.

Catatan Penting

  • Kualitas Gambar:
    • Gambar harus memiliki teks yang jelas, kontras tinggi, dan resolusi memadai untuk hasil OCR yang baik.
    • Teks yang buram, terdistorsi, atau dalam font yang tidak standar mungkin sulit dikenali.
  • Pemotongan:
    • Pemotongan dapat meningkatkan akurasi OCR dengan mengurangi noise dari area non-teks.
    • Pastikan kotak pemotongan mencakup semua teks yang ingin diekstrak.
  • Performa:
    • Proses OCR bergantung pada Tesseract.js, yang berjalan di sisi klien. Gambar besar atau kompleks mungkin memakan waktu lebih lama.
    • Tidak diperlukan koneksi internet setelah CDN dimuat, kecuali untuk mengunggah gambar dari URL (tidak didukung dalam versi ini).
  • Browser:
    • Gunakan browser modern untuk memastikan kompatibilitas dengan Vue.js 2, Tesseract.js, dan Cropper.js.
    • Beberapa browser lama mungkin tidak mendukung fitur seperti FileReader atau Cropper.js.
  • Keterbatasan:
    • Aplikasi tidak menyimpan gambar atau teks setelah sesi browser berakhir.
    • Tidak ada fitur untuk mengedit teks hasil OCR secara langsung di aplikasi.
